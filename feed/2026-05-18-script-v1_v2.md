# Apps Script v1 Discovery Document Gutted

**Date:** 2026-05-18  
**API:** `script.v1`  
**Impact:** High  
**⚠️ Breaking change**  

## Summary

The discovery document for the Apps Script API (v1) has been stripped of nearly all metadata, including OAuth scopes and global parameters.

## Details

A massive sweep has removed 868 elements from the script.v1 discovery doc. This includes all OAuth2 scope definitions (Gmail, Drive, Calendar, etc.), global query parameters like 'key', 'oauth_token', and 'fields', and core API metadata such as 'mtlsRootUrl' and 'canonicalName'. This level of removal typically indicates that this version of the discovery document is being retired or the API surface is being fundamentally restructured.
