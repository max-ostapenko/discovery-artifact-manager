# Iceberg Support for Timestamp Autodetection Precision

**Date:** 2026-05-30  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery has updated documentation to confirm Apache Iceberg support for timestamp precision autodetection and clarified Python UDF concurrency settings.

## Details

The `timestampTargetPrecision` field within `ExternalDataConfiguration` and `JobConfigurationLoad` now explicitly includes 'Iceberg External Table' as a supported format for autodetection mode. This allows developers to control how sub-second precision is handled when loading or querying Iceberg data. Additionally, the description for `ExternalRuntimeOptions.containerRequestConcurrency` was updated to specify it applies to Python UDF container instances, providing better context for developers tuning user-defined function performance.
