# Iceberg support for timestamp autodetection

**Date:** 2026-05-30  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery now supports timestamp precision autodetection for Iceberg External Tables and clarifies concurrency settings for Python UDFs.

## Details

The `timestampTargetPrecision` field within both `ExternalDataConfiguration` and `JobConfigurationLoad` has been updated to include support for the Iceberg External Table format, alongside CSV, Parquet, and Avro. Additionally, the description for `containerRequestConcurrency` in `ExternalRuntimeOptions` was updated to specify it applies to Python UDF container instances, providing better context for developers using remote functions.
