# Dataform introduces GitRepositoryLink for credentials

**Date:** 2026-06-18  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now supports gitRepositoryLink, enabling the use of managed machine credentials for Git operations.

## Details

A new optional field, `gitRepositoryLink`, has been added to the `GitRemoteSettings` schema. This field allows developers to specify a resource name (following the pattern `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) to handle Git authentication. This likely points toward integration with Google Cloud's managed connection services, providing a more secure alternative to manual credential management.
