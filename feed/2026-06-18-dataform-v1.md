# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-18  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform now supports referencing managed GitRepositoryLink resources for authentication, streamlining how machine credentials are used in SQL workflows.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field accepts a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*. This addition allows developers to leverage Google Cloud's managed repository connections for machine-based Git operations, providing a more secure and integrated alternative to manual credential management within Dataform.
