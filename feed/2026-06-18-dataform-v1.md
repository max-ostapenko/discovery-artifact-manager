# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-18  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking to managed Git repository connections for machine-based authentication via the new gitRepositoryLink field.

## Details

The GitRemoteSettings schema now includes an optional gitRepositoryLink field. This allows developers to point to a specific Git repository connection resource (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*) to handle machine credentials. This update facilitates more secure, centralized credential management for Git operations within Dataform, likely integrating with Google Cloud's Developer Connect or similar connection services.
