# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-18  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform's GitRemoteSettings now supports a gitRepositoryLink field for referencing managed Git connections.

## Details

The GitRemoteSettings schema has been updated to include a new optional field, gitRepositoryLink. This field allows developers to specify a resource name (following the format projects/*/locations/*/connections/*/gitRepositoryLinks/*) to be used for machine credentials, likely integrating with Google Cloud's Developer Connect or similar managed connection services.
