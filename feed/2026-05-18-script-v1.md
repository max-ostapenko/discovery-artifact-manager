# Apps Script API v1 Discovery Document Gutted

**Date:** 2026-05-18  
**API:** `script.v1`  
**Impact:** High  
**⚠️ Breaking change**  

## Summary

The discovery document for the Google Apps Script API (v1) has been effectively hollowed out, removing core metadata and all OAuth2 scope definitions.

## Details

A massive update to the discovery document has removed 868 entries, including the API's name, canonical name, and global parameters such as access_token, fields, and key. Crucially, the descriptions for all OAuth2 scopes—ranging from Gmail and Drive to internal Script project management—have been deleted. This level of removal typically indicates the API version is being decommissioned or significantly restructured in the discovery service, which will break automated client generators and documentation tools.
