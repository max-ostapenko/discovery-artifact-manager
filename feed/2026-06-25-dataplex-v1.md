# Dataplex introduces Data Domains for logical organization

**Date:** 2026-06-25  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex adds a new DataDomain resource type, enabling developers to create, list, and manage logical data boundaries within a project and location.

## Details

The API now includes a full suite of methods for the 'dataDomains' resource: 'create', 'get', 'list', and 'delete'. Creation and deletion are handled via Long Running Operations (LROs). The 'create' method supports a 'validateOnly' flag for dry-runs, while the 'list' method supports standard filtering and ordering. This addition facilitates better data mesh architectures by allowing formal domain definitions within the Dataplex ecosystem.
