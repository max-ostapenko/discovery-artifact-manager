# BigQuery adds Iceberg support for timestamp autodetection

**Date:** 2026-06-05  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery now supports timestamp precision autodetection for Iceberg External Tables and clarifies concurrency settings for Python UDFs.

## Details

The `timestampTargetPrecision` field within `ExternalDataConfiguration` and `JobConfigurationLoad` has been updated to include support for Iceberg External Tables, joining CSV, PARQUET, and AVRO. This allows developers to control how subsecond precision is handled during schema autodetection for Iceberg sources. Additionally, the description for `ExternalRuntimeOptions.containerRequestConcurrency` was clarified to specify it applies to Python UDF container instances.
