# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-10  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex now supports profiling unstructured data and inferring graph-based relationships between data entities via new scan types and schemas.

## Details

This update introduces the `UNSTRUCTURED_DATA_PROFILE` scan type to both `DataScan` and `DataScanJob` resources, accompanied by new specification and result schemas. A significant addition is the `GraphProfile` schema suite, which enables semantic inference of data relationships, including node types, edge types, and foreign key constraints. Additionally, the `LookupContextRequest` now includes an `all_schema_fields` option, allowing developers to retrieve full schema metadata regardless of the defined `context_budget`.
