# Unstructured Data Profiling and Graph Semantic Inference

**Date:** 2026-06-10  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex adds support for unstructured data profile scans and introduces a comprehensive graph profiling schema for semantic inference.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE type, allowing developers to configure scans via unstructuredDataProfileSpec and retrieve results through unstructuredDataProfileResult. A significant addition is the GraphProfile schema suite, which enables modeling complex data relationships including node types, edge types, and foreign key constraints for semantic inference. Additionally, LookupContextRequest now includes an all_schema_fields option to prevent the truncation of schema fields in the output context.
