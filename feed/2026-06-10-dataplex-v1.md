# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-10  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its data scanning capabilities with a new scan type for unstructured data and a comprehensive GraphProfile schema for semantic inference.

## Details

Dataplex has introduced support for profiling unstructured data. The `DataScan` and `DataScanJob` resources now include a `UNSTRUCTURED_DATA_PROFILE` type, accompanied by `unstructuredDataProfileSpec` for configuration and `unstructuredDataProfileResult` for results. Additionally, a large set of schemas for `GraphProfile` has been added, allowing the API to represent complex semantic relationships through `nodeTypes`, `edgeTypes`, and `foreignKeys`. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to force the return of all schema fields regardless of the `context_budget` setting.
