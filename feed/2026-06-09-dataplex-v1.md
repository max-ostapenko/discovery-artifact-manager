# Dataplex adds Unstructured Data Profiling and Graph Analysis

**Date:** 2026-06-09  
**API:** `dataplex.v1`  
**Impact:** High  

## Summary

Dataplex V1 introduces support for unstructured data profile scans and a comprehensive graph profiling schema for semantic inference.

## Details

The DataScan resource now supports a new UNSTRUCTURED_DATA_PROFILE type, accompanied by UnstructuredDataProfileSpec and UnstructuredDataProfileResult. Additionally, a massive set of schemas for GraphProfile has been added, enabling the representation of nodes, edges, and foreign key constraints for semantic data mapping. Finally, LookupContextRequest adds an all_schema_fields option to ensure full schema retrieval regardless of character budget.
