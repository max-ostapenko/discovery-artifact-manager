# Dataplex adds Unstructured Data Profiling and Graph Metadata

**Date:** 2026-06-09  
**API:** `dataplex.v1`  
**Impact:** High  

## Summary

Dataplex expands its scanning capabilities with a new Unstructured Data Profile scan type and a comprehensive Graph Profile schema for semantic inference.

## Details

The DataScan and DataScanJob resources now support a new scan type: UNSTRUCTURED_DATA_PROFILE. This comes with associated specification and result schemas (unstructuredDataProfileSpec and unstructuredDataProfileResult). Additionally, a large set of schemas for GraphProfile has been introduced, including NodeTypes, EdgeTypes, and ForeignKeys, allowing developers to represent and retrieve complex semantic relationships discovered during data scans. Finally, LookupContextRequest now includes an all_schema_fields option to prevent truncation of schema fields in the output context.
