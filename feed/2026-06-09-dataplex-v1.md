# Dataplex adds Unstructured Data Profiling and Graph Metadata

**Date:** 2026-06-09  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with support for unstructured data profiling and a new graph-based semantic inference model.

## Details

The DataScan and DataScanJob resources now support a new scan type: `UNSTRUCTURED_DATA_PROFILE`. This includes the addition of `unstructuredDataProfileSpec` for configuration and `unstructuredDataProfileResult` for output. Furthermore, a massive set of schemas has been added to support Graph Profiling (e.g., `GoogleCloudDataplexV1GraphProfile`, `GraphProfileEdgeType`, `GraphProfileNodeType`), enabling semantic inference of relationships between data entities. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to bypass context budget limits and return full schema metadata.
