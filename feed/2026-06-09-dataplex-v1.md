# Dataplex adds Unstructured Data Profiling and Graph Discovery

**Date:** 2026-06-09  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its data scanning capabilities with a new unstructured data profile scan type and comprehensive graph profiling for semantic inference.

## Details

Dataplex has introduced support for profiling unstructured data. The `DataScan` and `DataScanJob` resources now include a `UNSTRUCTURED_DATA_PROFILE` type, along with `unstructuredDataProfileSpec` for configuration and `unstructuredDataProfileResult` for output. Additionally, a large set of schemas for Graph Profiling has been added (e.g., `GraphProfile`, `GraphProfileEdgeType`, `GraphProfileNodeType`), allowing the service to infer and describe complex relationships, cardinalities, and foreign key constraints in data. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to prevent truncation of schema fields in the response.
