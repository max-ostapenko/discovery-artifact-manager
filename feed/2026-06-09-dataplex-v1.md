# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-09  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex DataScan now supports unstructured data profiling and introduces a comprehensive graph-based semantic inference model for data relationships.

## Details

This update introduces the `UNSTRUCTURED_DATA_PROFILE` scan type to `DataScan` and `DataScanJob` resources, accompanied by new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` schemas. Additionally, a large set of Graph Profile schemas has been added (including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`), enabling the API to represent complex data relationships, edge types, and foreign key constraints discovered during semantic inference. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to prevent truncation of schema fields in the context output.
