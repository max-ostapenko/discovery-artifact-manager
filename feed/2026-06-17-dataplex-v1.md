# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-17  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex V1 introduces support for unstructured data profile scans and a comprehensive graph profiling schema for semantic inference.

## Details

This update adds a major new scan type, UNSTRUCTURED_DATA_PROFILE, to the DataScan and DataScanJob resources, accompanied by new Spec and Result schemas. Furthermore, a suite of GraphProfile schemas has been introduced (including NodeTypes, EdgeTypes, and ForeignKeys) to support semantic inference scan results. Developers using LookupContextRequest also gain a new 'all_schema_fields' option to ensure full schema field output regardless of the context budget.
