# Dataplex adds unstructured data profiling and graph scans

**Date:** 2026-06-17  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with support for unstructured data profiling and a new graph-based profiling structure for semantic inference.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE type, accompanied by specific spec and result schemas (unstructuredDataProfileSpec and unstructuredDataProfileResult). Additionally, a large set of schemas for GraphProfile has been introduced, including EdgeTypes and NodeTypes, which allow for detailed semantic inference scan results that describe relationships and entities in a graph format. Finally, LookupContextRequest now includes an all_schema_fields option to force the return of all schema fields regardless of the character budget.
