# Dataplex adds Unstructured Data Profiling and Graph Analysis

**Date:** 2026-06-17  
**API:** `dataplex.v1`  
**Impact:** High  

## Summary

Dataplex V1 introduces support for scanning unstructured data and a comprehensive graph profiling schema for semantic inference and relationship mapping.

## Details

This update significantly expands Dataplex's scanning capabilities. A new scan type, `UNSTRUCTURED_DATA_PROFILE`, has been added to `DataScan` and `DataScanJob` resources, accompanied by `unstructuredDataProfileSpec` for configuration and `unstructuredDataProfileResult` for results. Additionally, a robust set of schemas for Graph Profiling has been introduced (including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`), enabling developers to perform semantic inference scans that identify nodes, edges, and foreign key relationships across data assets. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to bypass truncation and return full schema metadata.
