# Iceberg support added to timestamp autodetection

**Date:** 2026-06-09  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery documentation now reflects support for Iceberg External Tables in timestamp precision autodetection and clarifies Python UDF concurrency settings.

## Details

The `timestampTargetPrecision` field in `ExternalDataConfiguration` and `JobConfigurationLoad` now includes Iceberg External Tables alongside CSV, PARQUET, and AVRO for autodetection mode. Additionally, the description for `ExternalRuntimeOptions.containerRequestConcurrency` was updated to specify it applies to Python UDF container instances rather than generic Cloud Run instances.
