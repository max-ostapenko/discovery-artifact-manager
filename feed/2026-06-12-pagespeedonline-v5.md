# New Agentic Browsing Audits in PageSpeed Insights

**Date:** 2026-06-12  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now includes a new 'agentic-browsing' category in its Lighthouse reports, enabling audits focused on AI agent interactions.

## Details

A new property `agentic-browsing` (referencing `LighthouseCategoryV5`) has been added to the `Categories` schema. This category is intended to group audits specifically related to agentic browsing, suggesting new metrics for how well a page supports automated or AI-driven browsing agents.
