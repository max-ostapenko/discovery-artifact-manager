# PageSpeed Insights adds Agentic Browsing category

**Date:** 2026-06-12  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

A new Lighthouse category for 'agentic browsing' has been introduced, targeting how AI agents and automated browsers experience web content.

## Details

The Categories schema now includes an `agentic-browsing` property, which maps to a `LighthouseCategoryV5` object. This addition allows developers to retrieve specific audits focused on agentic interactions, signaling a shift in how Google evaluates page performance for non-human consumers like AI agents.
