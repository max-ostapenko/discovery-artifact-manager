# PageSpeed Insights adds 'Agentic Browsing' category

**Date:** 2026-06-12  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

A new audit category for 'agentic browsing' has been added to the PageSpeed Insights API, signaling a focus on how AI agents interact with web content.

## Details

The Categories schema has been updated to include a new property, agentic-browsing, which utilizes the LighthouseCategoryV5 type. This addition suggests that Google is beginning to track and report on how well websites perform or provide data when accessed by automated AI agents or LLM-based browsers, rather than just human users.
