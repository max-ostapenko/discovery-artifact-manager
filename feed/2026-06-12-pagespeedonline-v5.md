# New 'Agentic Browsing' category added to PageSpeed audits

**Date:** 2026-06-12  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

PageSpeed Insights now includes a new 'agentic-browsing' category in its Lighthouse reports, targeting automated and AI-driven browsing scenarios.

## Details

The Categories schema has been updated to include a new property: 'agentic-browsing'. This category, which uses the LighthouseCategoryV5 schema, is designed to group audits specifically related to how 'agentic' browsers (likely AI agents or automated web navigators) interact with and perceive a webpage. This addition suggests a shift toward optimizing web content for non-human consumers.
