# New 'Agentic Browsing' category added to Lighthouse audits

**Date:** 2026-06-12  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

PageSpeed Insights now includes a dedicated category for agentic browsing audits, allowing developers to measure page compatibility with automated AI agents.

## Details

The Categories schema has been updated to include a new property: agentic-browsing. This field uses the LighthouseCategoryV5 schema and is designed to group audits related to how well a page performs when accessed by 'agentic' or automated browsing tools. This addition suggests a shift in how Google evaluates web content for AI-driven consumption.
