# PageSpeed Insights adds Agentic Browsing category

**Date:** 2026-06-18  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

The PageSpeed Insights API now includes a new category for 'agentic browsing' audits, hinting at performance metrics for automated agents.

## Details

The Categories schema has been updated to include a new property, agentic-browsing, which maps to the LighthouseCategoryV5 type. This addition suggests that Lighthouse is beginning to track and report on audits specifically relevant to how automated or AI-driven agents interact with web pages, expanding the report surface beyond traditional human-centric performance and accessibility metrics.
