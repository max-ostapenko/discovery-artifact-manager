# New Agentic Browsing Category in PageSpeed Audits

**Date:** 2026-06-18  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now includes a new 'agentic-browsing' category in Lighthouse reports, grouping audits related to AI-driven or agentic web interactions.

## Details

A new property 'agentic-browsing' has been added to the 'Categories' schema, referencing 'LighthouseCategoryV5'. This addition signals a new focus on measuring how websites perform and behave when accessed by automated AI agents rather than traditional human-driven browsers. Developers can expect new audits to populate this category as Google expands its tooling for the agentic web.
