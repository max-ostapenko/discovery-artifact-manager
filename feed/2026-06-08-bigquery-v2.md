# Iceberg support and Python UDF documentation updates

**Date:** 2026-06-08  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery documentation now explicitly includes Iceberg External Tables for timestamp precision settings and clarifies Python UDF concurrency options.

## Details

The timestampTargetPrecision field in both ExternalDataConfiguration and JobConfigurationLoad has been updated to include 'Iceberg External Table' as a supported format for autodetection mode. Additionally, ExternalRuntimeOptions.containerRequestConcurrency has been clarified to refer specifically to Python UDF container instances rather than generic Cloud Run instances, providing better context for developers using remote functions.
