# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-13  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with support for unstructured data profiling and complex graph-based semantic inference.

## Details

This update introduces the `UNSTRUCTURED_DATA_PROFILE` scan type to `DataScan` and `DataScanJob` resources, enabling profiling of non-tabular data. Additionally, a large set of schemas has been added to support Graph Profiling (`GraphProfile`), which allows for the discovery of semantic relationships, node types, and edge types (including foreign key constraints and cardinality hints) within data. Finally, `LookupContextRequest` now includes an `all_schema_fields` option to ensure full schema visibility regardless of the defined `context_budget`.
