# Dataplex adds Unstructured Data Profiling and Graph Analysis

**Date:** 2026-06-13  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with support for unstructured data profiling and a new graph-based semantic inference model.

## Details

Dataplex has introduced a major expansion to its DataScan resource. A new scan type, `UNSTRUCTURED_DATA_PROFILE`, allows developers to profile non-tabular data sources. Accompanying this is a robust set of Graph Profile schemas (including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`) designed for semantic inference, enabling the analysis of relationships, edge cardinality, and foreign key constraints within data ecosystems. Additionally, the `LookupContextRequest` now supports an `all_schema_fields` option to force the return of all schema fields regardless of the context budget.
