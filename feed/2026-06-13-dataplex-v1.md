# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-13  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its DataScan capabilities with support for unstructured data profiling and a new graph-based semantic inference model.

## Details

The DataScan and DataScanJob resources now support a new scan type: `UNSTRUCTURED_DATA_PROFILE`. This includes the addition of `unstructuredDataProfileSpec` for configuration and `unstructuredDataProfileResult` for output. Furthermore, a large set of schemas for `GraphProfile` has been introduced, allowing the API to describe inferred semantic relationships between data entities, including node types, edge types, and foreign key constraints. Additionally, `LookupContextRequest` now supports an `all_schema_fields` option to bypass truncation when retrieving schema metadata.
