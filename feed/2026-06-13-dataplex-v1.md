# Dataplex adds Unstructured Data Profiling and Graph Analysis

**Date:** 2026-06-13  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex DataScan now supports unstructured data profiling and introduces a comprehensive graph-based semantic inference model.

## Details

The DataScan resource has been expanded with a new 'UNSTRUCTURED_DATA_PROFILE' type, supported by the addition of UnstructuredDataProfileSpec and UnstructuredDataProfileResult schemas. This allows developers to profile non-tabular data sources. Furthermore, a large set of schemas for Graph Profiling has been introduced (GraphProfile, GraphProfileEdgeType, GraphProfileNodeType), enabling the capture of complex relationships, cardinalities, and foreign key constraints within data lakes. Additionally, LookupContextRequest now supports an 'all_schema_fields' option to prevent truncation of schema metadata.
