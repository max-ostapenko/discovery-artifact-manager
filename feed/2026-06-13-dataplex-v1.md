# Dataplex adds Unstructured Data Profiling and Graph Scans

**Date:** 2026-06-13  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new UNSTRUCTURED_DATA_PROFILE scan type and comprehensive graph profiling schemas for semantic inference.

## Details

Developers can now configure DataScan and DataScanJob resources with the new UNSTRUCTURED_DATA_PROFILE type, supported by the added unstructuredDataProfileSpec and unstructuredDataProfileResult fields. Additionally, a significant set of schemas for GraphProfile has been introduced, enabling detailed semantic inference results including node types, edge types, and foreign key constraints. Finally, the LookupContextRequest now supports an all_schema_fields option to prevent truncation of schema fields in the output context.
