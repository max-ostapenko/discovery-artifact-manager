# Looker v1 adds Instance Undelete and Release Channels

**Date:** 2026-06-17  
**API:** `looker.v1`  
**Impact:** Medium  

## Summary

Looker Core now supports soft-deletion recovery and configurable release channels (Rapid, Regular, Stable). Developers also gain control over accelerated security patches and can list previously deleted instances.

## Details

A new `undelete` method has been added to the `instances` resource, complemented by a `showDeleted` boolean query parameter in the `list` method. The `Instance` schema now includes `releaseChannel` for update cadence management, `acceleratedSecurityPatchEnabled` for faster security rollouts, and metadata fields like `softDeleteReason` and `suspendedTime` to track instance lifecycle states. Additionally, platform edition descriptions have been clarified to reflect specific Looker Core tiers.
