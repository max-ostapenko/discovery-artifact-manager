# Soft-delete recovery and release channels for Looker

**Date:** 2026-06-17  
**API:** `looker.v1`  
**Impact:** Medium  

## Summary

Looker Core adds instance recovery via a new undelete method and introduces release channel management for update frequency control.

## Details

A new `undelete` method and `showDeleted` list parameter enable recovery of instances deleted by users, billing issues, or trial expirations. Additionally, the `Instance` resource now supports `releaseChannel` (RAPID, REGULAR, STABLE) and `acceleratedSecurityPatchEnabled` for better control over maintenance and security updates. The API also adds metadata fields like `softDeleteReason` and `suspendedTime` to help track the lifecycle of deleted instances.
