# Looker v1 adds Instance Undelete and Release Channels

**Date:** 2026-06-17  
**API:** `looker.v1`  
**Impact:** Medium  

## Summary

Looker Core now supports soft-delete recovery and configurable release channels for instance updates.

## Details

This update introduces a safety net for Looker instances with the new `instances.undelete` method and a `showDeleted` filter for the `list` method. Developers can now programmatically manage instance update cadences via the new `releaseChannel` field (supporting RAPID, REGULAR, and STABLE) and toggle `acceleratedSecurityPatchEnabled`. Additionally, new output-only fields like `softDeleteReason` and `suspendedTime` provide better visibility into instance suspension causes, such as billing issues or trial expirations.
