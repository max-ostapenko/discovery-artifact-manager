# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-11  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking GitRepositoryLink resources within GitRemoteSettings, streamlining machine-based authentication for repositories.

## Details

The GitRemoteSettings schema includes a new optional gitRepositoryLink field. This allows developers to reference a specific Git repository connection resource (format: projects/*/locations/*/connections/*/gitRepositoryLinks/*) instead of relying solely on traditional credential methods. This is particularly useful for managing machine credentials in automated workflows and CI/CD pipelines.
