# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-11  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking to managed GitRepositoryLink resources for authentication, enabling more secure and centralized machine credential management.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field allows developers to specify a resource name (following the format projects/*/locations/*/connections/*/gitRepositoryLinks/*) to handle Git authentication. This shift suggests integration with Google Cloud's broader repository connection framework, potentially moving away from manual secret handling for Git operations.
