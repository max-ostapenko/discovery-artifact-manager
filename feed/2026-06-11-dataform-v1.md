# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-11  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking to GitRepositoryLink resources for machine credentials, streamlining how repositories authenticate with remote Git providers.

## Details

The `GitRemoteSettings` schema has been updated to include a new optional field: `gitRepositoryLink`. This field allows developers to specify a resource name (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) to handle machine credentials. This is a significant improvement for teams looking to move away from manual secret management in favor of Google Cloud's managed Git connections.
