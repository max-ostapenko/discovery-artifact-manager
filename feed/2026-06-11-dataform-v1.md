# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-11  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now supports gitRepositoryLink, enabling the use of managed Google Cloud connection resources for Git authentication.

## Details

The GitRemoteSettings schema has been updated with a new optional string field: `gitRepositoryLink`. This field accepts a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`. This addition allows developers to leverage Google Cloud's managed connectivity services (like Developer Connect) for machine credentials, providing a more structured and secure alternative to manual secret management for Git-based SQL workflows.
