# PageSpeed Insights adds Agentic Browsing category

**Date:** 2026-06-21  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now includes a new 'agentic-browsing' category in its Lighthouse reports, targeting AI-driven web interaction.

## Details

A new property `agentic-browsing` has been added to the `Categories` schema, referencing the `LighthouseCategoryV5` type. This addition introduces a specific category for audits related to how AI agents or automated browsers interact with and perceive web content, reflecting a shift toward optimizing for non-human web consumers.
