# PageSpeed Insights adds Agentic Browsing category

**Date:** 2026-06-21  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now includes a new 'agentic-browsing' category for Lighthouse audits, targeting automated and AI-driven browsing performance.

## Details

A new property `agentic-browsing` has been added to the `Categories` schema, referencing the `LighthouseCategoryV5` type. This addition suggests a new suite of audits specifically designed to measure how well a page handles 'agentic' traffic—likely referring to LLM-based agents, automated browsers, or AI crawlers that interact with web content differently than traditional human users.
