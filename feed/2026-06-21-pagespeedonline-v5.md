# New Agentic Browsing Category Added to Lighthouse

**Date:** 2026-06-21  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

The PageSpeed Insights API has introduced a new Lighthouse category specifically for 'agentic browsing' audits.

## Details

A new property, `agentic-browsing`, has been added to the `Categories` schema, referencing the `LighthouseCategoryV5` type. This suggests that Google is beginning to track and audit how websites perform or behave when accessed by autonomous agents or AI-driven browsers, alongside traditional metrics like performance and SEO.
