# New Agentic Browsing category added to Lighthouse audits

**Date:** 2026-06-21  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now includes a new audit category for 'agentic browsing,' signaling a shift toward measuring how well sites perform for automated AI agents.

## Details

A new property `agentic-browsing` has been added to the `Categories` schema, referencing the `LighthouseCategoryV5` type. This category is intended to house audits specifically related to agentic browsing behaviors, likely providing insights into how AI-driven crawlers and automated agents perceive and interact with web content.
