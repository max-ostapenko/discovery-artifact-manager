# New Agentic Browsing Category Added to Lighthouse

**Date:** 2026-06-21  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

The PageSpeed Insights API now includes a new 'agentic-browsing' category in its Lighthouse audit results.

## Details

A new property `agentic-browsing` has been added to the `Categories` schema, referencing the `LighthouseCategoryV5` type. This category is designed to house audits specifically related to agentic browsing, signaling a shift toward measuring how well web pages perform for AI-driven agents and automated browsers.
