# New Agentic Browsing Category in Lighthouse Results

**Date:** 2026-06-21  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

The PageSpeed Insights API now includes a new 'agentic-browsing' category within its Lighthouse audit results.

## Details

A new property, `agentic-browsing`, has been added to the `Categories` schema. This category, which maps to the `LighthouseCategoryV5` type, is designed to aggregate audits related to how automated agents or AI-driven browsers interact with and perceive web pages. This addition signals a shift toward measuring and optimizing web performance for non-human visitors.
