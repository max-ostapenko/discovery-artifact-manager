# New 'Agentic Browsing' Category Added to Lighthouse Audits

**Date:** 2026-06-21  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now includes a dedicated 'agentic-browsing' category in its Lighthouse results, targeting automated or AI-driven browsing scenarios.

## Details

A new property `agentic-browsing` has been added to the `Categories` schema, referencing the `LighthouseCategoryV5` type. This addition allows developers to programmatically access a new suite of audits focused on 'agentic' interactions—likely reflecting how AI agents and automated tools perceive and interact with web page performance and accessibility.
