# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-13  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports GitRepositoryLink within GitRemoteSettings, allowing for more standardized machine credential management for Git remotes.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field accepts a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*. This change enables developers to link Dataform workspaces to Git repositories using managed connections rather than relying solely on manual credential configurations.
