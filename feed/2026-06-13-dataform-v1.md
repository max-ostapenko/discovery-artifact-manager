# Dataform adds Developer Connect support for Git

**Date:** 2026-06-13  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports gitRepositoryLink in GitRemoteSettings, enabling machine credentials via Google Cloud Developer Connect.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field allows developers to link Dataform to a Git repository using a resource name from the Developer Connect service (format: projects/*/locations/*/connections/*/gitRepositoryLinks/*). This is a significant security improvement for automated workflows, as it moves away from manual secret management toward managed machine credentials.
