# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-17  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now includes a gitRepositoryLink field, enabling the use of machine credentials via centralized connection resources.

## Details

The addition of `gitRepositoryLink` to the `GitRemoteSettings` schema allows developers to point to a specific `GitRepositoryLink` resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`). This update facilitates better integration with Google Cloud's developer connection infrastructure for managing Git authentication, providing a more standardized way to handle machine-to-machine credentials for repository access.
