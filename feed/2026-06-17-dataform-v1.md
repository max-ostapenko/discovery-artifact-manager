# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-17  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform's GitRemoteSettings now supports a gitRepositoryLink field for managing machine-based Git authentication via resource names.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field allows developers to specify a resource name (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*) to handle machine credentials. This likely facilitates better integration with Google Cloud's connectivity services for more secure, centralized Git authentication compared to manual token management.
