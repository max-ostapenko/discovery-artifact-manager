# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-17  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking Git repositories via a dedicated GitRepositoryLink resource, enabling better management of machine credentials.

## Details

The `GitRemoteSettings` schema has been updated with a new optional field: `gitRepositoryLink`. This field allows developers to specify a resource name (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) to handle authentication. This shift suggests a move toward using Google Cloud's managed connection services for Git integration rather than relying solely on manual secret handling.
