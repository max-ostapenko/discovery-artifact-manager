# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-17  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform's Git settings now support gitRepositoryLink, enabling a more structured way to manage machine credentials via external connection resources.

## Details

The GitRemoteSettings schema has been updated with an optional gitRepositoryLink field. This field accepts a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*. This addition facilitates the use of centralized connection management for Git authentication, allowing developers to move away from manually managed tokens toward more robust machine-to-machine credentialing.
