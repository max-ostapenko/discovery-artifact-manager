# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-04  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform's Git settings now support a managed `gitRepositoryLink` field, enabling more structured machine-to-machine authentication.

## Details

The `GitRemoteSettings` schema has been updated with an optional `gitRepositoryLink` string field. This field allows developers to reference a specific Git connection resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) to handle machine credentials, likely streamlining integration with Google Cloud's managed developer connection services.
