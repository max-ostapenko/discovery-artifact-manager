# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-09  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports GitRepositoryLink resources for machine credentials in Git remote settings, enabling integration with managed repository connections.

## Details

The GitRemoteSettings schema now includes an optional gitRepositoryLink field. This field accepts a resource name (projects/*/locations/*/connections/*/gitRepositoryLinks/*), allowing Dataform to leverage Developer Connect repository links for authentication. This provides a more managed way to handle machine credentials compared to manual token or SSH key management.
