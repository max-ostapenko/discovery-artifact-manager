# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-09  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform's Git configuration now supports a direct link to GitRepositoryLink resources, simplifying machine-based authentication.

## Details

The GitRemoteSettings schema has been updated to include a new optional field, gitRepositoryLink. This field allows developers to specify a resource name (following the format projects/*/locations/*/connections/*/gitRepositoryLinks/*) to manage machine credentials, likely integrating with Google Cloud's broader connection management infrastructure.
