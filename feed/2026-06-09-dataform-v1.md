# Dataform adds managed GitRepositoryLink support

**Date:** 2026-06-09  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` in `GitRemoteSettings`, enabling the use of managed machine credentials for Git connections.

## Details

A new optional field `gitRepositoryLink` has been added to the `GitRemoteSettings` schema. This allows developers to point to a specific `GitRepositoryLink` resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) for authentication. This change facilitates the use of Google Cloud's managed connection infrastructure for Git operations, reducing the need for manual secret management for SSH keys or Personal Access Tokens.
