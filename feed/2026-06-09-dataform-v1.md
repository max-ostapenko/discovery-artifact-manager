# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-09  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` in Git remote settings, allowing developers to use managed repository connections for machine authentication.

## Details

The `GitRemoteSettings` schema has been updated with a new optional field: `gitRepositoryLink`. This field accepts a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`. This change facilitates the use of machine credentials via GCP's managed connection services, streamlining how Dataform interacts with remote Git repositories and improving security posture by leveraging centralized connection management.
