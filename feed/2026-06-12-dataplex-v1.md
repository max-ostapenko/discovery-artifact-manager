# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-12  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex is expanding beyond structured data with new support for unstructured data profiling and semantic graph inference.

## Details

This update introduces the `UNSTRUCTURED_DATA_PROFILE` scan type to `DataScan` and `DataScanJob` resources, allowing developers to profile non-tabular data sources. Additionally, a massive set of schemas for `GraphProfile` has been added, enabling semantic inference scans that identify node types, edge types, and foreign key relationships within data. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to bypass truncation and return complete schema metadata.
