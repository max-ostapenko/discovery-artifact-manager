# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-12  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex now supports unstructured data profile scans and introduces a comprehensive schema for graph-based semantic inference and profiling.

## Details

The DataScan and DataScanJob resources have been expanded with a new UNSTRUCTURED_DATA_PROFILE type, including specific spec and result fields for profiling unstructured content. Additionally, a large set of new schemas related to GraphProfile have been introduced (e.g., GraphProfileEdgeType, GraphProfileNodeType), allowing for the representation of complex relationship topologies, edge cardinalities, and foreign key constraints. Finally, the LookupContextRequest now includes an all_schema_fields option to ensure full schema retrieval regardless of the context budget.
