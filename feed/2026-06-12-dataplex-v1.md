# Dataplex adds Unstructured Data Profiling and Graph Metadata

**Date:** 2026-06-12  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new unstructured data profile scan type and a comprehensive set of Graph Profile schemas for semantic inference.

## Details

Dataplex has introduced support for unstructured data profiling. The `DataScan` and `DataScanJob` resources now include a `UNSTRUCTURED_DATA_PROFILE` enum and associated `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. Additionally, a large set of new schemas related to `GraphProfile` have been added (including `GraphProfileNodeType`, `GraphProfileEdgeType`, and `GraphProfileField`), enabling the API to represent complex semantic relationships and graph-based data structures. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to force the return of all schema fields regardless of the defined context budget.
