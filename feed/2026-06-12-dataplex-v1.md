# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-12  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its DataScan capabilities with a new UNSTRUCTURED_DATA_PROFILE type and comprehensive Graph Profiling schemas for semantic inference.

## Details

This update introduces support for profiling unstructured data via the new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields in the DataScan resource. Additionally, a large set of schemas has been added to support Graph Profiling, including `GraphProfile`, `GraphProfileNodeType`, and `GraphProfileEdgeType`, which allow for detailed semantic inference including edge cardinality and foreign key constraints. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to prevent truncation of schema fields in the output context.
