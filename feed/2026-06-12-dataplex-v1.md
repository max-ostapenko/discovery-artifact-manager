# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-12  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex introduces support for unstructured data profile scans and a comprehensive graph profiling schema for semantic inference.

## Details

The DataScan resource now supports a new UNSTRUCTURED_DATA_PROFILE type, accompanied by specific specification and result schemas (unstructuredDataProfileSpec and unstructuredDataProfileResult). A significant set of new schemas for GraphProfile has been added, including nodeTypes, edgeTypes, and foreignKeys, which facilitate detailed semantic inference of data relationships. Additionally, the LookupContextRequest now includes an all_schema_fields option to ensure all schema fields are returned regardless of the context budget.
