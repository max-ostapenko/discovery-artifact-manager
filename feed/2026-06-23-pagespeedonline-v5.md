# New Agentic Browsing category added to PageSpeed API

**Date:** 2026-06-23  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now supports a new audit category called AGENTIC_BROWSING to evaluate site compatibility with AI agents.

## Details

A new enum value `AGENTIC_BROWSING` has been added to the `category` parameter of the `pagespeedapi.runpagespeed` method. This category is designed to measure a website's ability to be rendered and navigated by agentic browsing systems, signaling a shift toward optimizing web content for AI-driven consumers rather than just human users.
