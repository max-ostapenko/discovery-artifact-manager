# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-11  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex has introduced support for profiling unstructured data and a new graph-based schema for semantic inference scan results.

## Details

The DataScan resource now supports a new enum type `UNSTRUCTURED_DATA_PROFILE`, enabling developers to configure and retrieve results for scans on non-tabular data via `unstructuredDataProfileSpec` and `unstructuredDataProfileResult`. Additionally, a comprehensive set of schemas for Graph Profiling has been added (including `GraphProfile`, `GraphProfileNodeType`, and `GraphProfileEdgeType`), which allows the API to represent complex semantic relationships, cardinalities, and foreign key constraints discovered during scans. Finally, `LookupContextRequest` now includes an `all_schema_fields` option to prevent truncation of schema fields regardless of the character budget.
