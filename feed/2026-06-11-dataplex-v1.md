# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-11  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new scan type for unstructured data and a comprehensive suite of schemas for graph-based semantic inference.

## Details

The API introduces the `UNSTRUCTURED_DATA_PROFILE` scan type to `DataScan` and `DataScanJob` resources, supported by new `UnstructuredDataProfileSpec` and `UnstructuredDataProfileResult` schemas. Furthermore, a large set of new schemas has been added to support graph profiling (e.g., `GraphProfile`, `GraphProfileEdgeType`, `GraphProfileNodeType`), enabling the discovery of semantic relationships, edge cardinalities, and foreign key constraints across data assets. Additionally, `LookupContextRequest` now supports an `all_schema_fields` option to return complete schema information regardless of the character budget.
