# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-11  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its DataScan capabilities with support for unstructured data profiling and graph-based semantic inference.

## Details

The DataScan and DataScanJob resources now include a new UNSTRUCTURED_DATA_PROFILE type, supported by the addition of UnstructuredDataProfileSpec and UnstructuredDataProfileResult schemas. Furthermore, a robust set of schemas for GraphProfile has been introduced, enabling the discovery of node and edge types, foreign key constraints, and cardinality hints within data. Additionally, the LookupContextRequest now supports an all_schema_fields option to bypass truncation when retrieving schema metadata.
