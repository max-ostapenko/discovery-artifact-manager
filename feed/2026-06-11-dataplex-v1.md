# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-11  
**API:** `dataplex.v1`  
**Impact:** High  

## Summary

Dataplex expands its scanning capabilities with support for unstructured data profiling and a new graph-based profile structure for semantic inference.

## Details

Developers can now utilize the `UNSTRUCTURED_DATA_PROFILE` scan type within `DataScan` and `DataScanJob` resources, which includes new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. Additionally, a comprehensive suite of graph profiling schemas has been introduced—including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`—enabling the analysis of relationships, cardinalities, and foreign keys. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to prevent truncation of schema fields in the output.
