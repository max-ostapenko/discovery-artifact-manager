# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-11  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its DataScan capabilities with a new scan type for unstructured data and a comprehensive Graph Profiling schema for semantic inference.

## Details

The API now supports `UNSTRUCTURED_DATA_PROFILE` as a valid type for `DataScan` and `DataScanJob` resources, accompanied by new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. Furthermore, a significant set of schemas has been added to support graph-based profiling, including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`, which allow for detailed mapping of relationships, edge cardinality, and foreign key constraints. Additionally, `LookupContextRequest` now supports an `all_schema_fields` option to prevent metadata truncation.
