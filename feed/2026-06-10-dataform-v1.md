# Dataform adds Developer Connect support for Git

**Date:** 2026-06-10  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's Git settings now include a gitRepositoryLink field, enabling machine credential integration via Google Cloud Developer Connect.

## Details

The GitRemoteSettings resource has been updated to include a new optional field: gitRepositoryLink. This field accepts a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*. This allows developers to leverage Google Cloud's Developer Connect service for Git authentication, providing a more secure and centralized way to manage machine credentials compared to manual secret management.
