# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-10  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` within `GitRemoteSettings`, allowing developers to use managed Git repository connections for machine-based authentication.

## Details

The `GitRemoteSettings` schema has been updated with a new optional `gitRepositoryLink` field. This field accepts a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`. This change simplifies how Dataform handles machine credentials by leveraging centralized Git repository links, facilitating better integration with Google Cloud's managed connectivity services.
