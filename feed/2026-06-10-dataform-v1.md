# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-10  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` in Git settings, enabling centralized credential management via Google Cloud's connection resources.

## Details

The `GitRemoteSettings` resource now includes an optional `gitRepositoryLink` field. This allows developers to reference a `GitRepositoryLink` resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) for machine credentials. This change facilitates better integration with Google Cloud's Developer Connect service, moving away from manual secret management for Git authentication.
