# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-15  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports gitRepositoryLink within GitRemoteSettings, enabling managed machine-based authentication for remote Git repositories.

## Details

The GitRemoteSettings schema has been updated with an optional gitRepositoryLink field. This field allows developers to reference a GitRepositoryLink resource (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*) for machine credentials, likely integrating with GCP's Developer Connect or similar connectivity services to streamline secure repository access.
