# Dataform adds Developer Connect support for Git

**Date:** 2026-06-15  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports gitRepositoryLink in GitRemoteSettings, enabling machine credentials via Google Cloud's Developer Connect.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field allows developers to link a Dataform repository to a GitRepositoryLink resource (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*). This integration facilitates more secure, centralized credential management for Git operations within Dataform.
