# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-15  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform now supports referencing GitRepositoryLink resources for Git authentication, enabling better integration with managed machine credentials.

## Details

A new optional field `gitRepositoryLink` has been added to the `GitRemoteSettings` schema. This field allows developers to specify a resource name (following the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) to handle Git credentials, likely leveraging Google Cloud's Developer Connect or similar connectivity services instead of manual secret management.
