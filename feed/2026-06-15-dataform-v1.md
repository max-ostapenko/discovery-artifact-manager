# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-15  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's Git settings now support gitRepositoryLink, allowing for more secure, managed authentication via Google Cloud's connection resources.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field allows developers to specify a resource name (following the format projects/*/locations/*/connections/*/gitRepositoryLinks/*) to handle machine credentials. This likely points toward integration with Developer Connect, simplifying how Dataform authenticates with remote repositories without requiring manual secret management for every workspace.
