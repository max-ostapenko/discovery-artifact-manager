# Documentation update for pipeline sources

**Date:** 2026-06-19  
**API:** `datapipelines.v1`  
**Impact:** Low  

## Summary

The description for pipeline sources has been updated to reference Knowledge Catalog instead of Dataplex.

## Details

This is a documentation-only change. The description for the `pipelineSources` field in the `GoogleCloudDatapipelinesV1Pipeline` schema has been modified to use 'Knowledge Catalog' as an example of a source, replacing the previous reference to 'Dataplex'. No functional changes were made to the API schema or logic.
