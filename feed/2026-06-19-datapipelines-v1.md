# Documentation update for pipeline sources

**Date:** 2026-06-19  
**API:** `datapipelines.v1`  
**Impact:** Low  

## Summary

The description for pipeline sources has been updated to reference Knowledge Catalog instead of Dataplex as an example.

## Details

This is a documentation-only change. The pipelineSources field in the GoogleCloudDatapipelinesV1Pipeline schema now lists 'Knowledge Catalog' as an example source in its description, replacing 'Dataplex'. There are no functional changes to the API surface or behavior.
