# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-20  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform's Git settings now support a gitRepositoryLink field, allowing developers to reference managed Git repository connections for machine-based authentication.

## Details

The GitRemoteSettings schema has been updated with a new optional string field: gitRepositoryLink. This field expects a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*. This change likely integrates Dataform more deeply with Google Cloud's secure connection management services, simplifying how machine credentials are handled for Git operations compared to manual token management.
