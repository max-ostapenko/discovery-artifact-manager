# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-20  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` in Git settings, enabling managed machine-based authentication for Git repositories.

## Details

The `GitRemoteSettings` resource has been updated with a new optional field: `gitRepositoryLink`. This field accepts a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`. This addition allows developers to leverage GCP-managed Git connections for machine credentials, moving away from manual secret management for automated SQL workflows.
