# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-20  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now includes a gitRepositoryLink field, enabling the use of managed connection resources for machine-based Git authentication.

## Details

The addition of `gitRepositoryLink` to the `GitRemoteSettings` schema allows developers to reference a specific resource name (following the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) for machine credentials. This provides a more structured and potentially more secure way to handle Git authentication by leveraging GCP's managed connection services rather than manual secret management.
