# Dataform adds Developer Connect support for Git

**Date:** 2026-06-20  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` in `GitRemoteSettings`, enabling machine credentials via GCP Developer Connect.

## Details

The `GitRemoteSettings` schema now includes an optional `gitRepositoryLink` field. This allows users to link Dataform to Git repositories using the standard GCP connection resource format (`projects/*/locations/*/connections/*/gitRepositoryLinks/*`). This move facilitates standardized, managed machine identities for Git operations within Dataform, moving away from manual secret handling.
