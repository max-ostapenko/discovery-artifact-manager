# New Agentic Browsing category added to PageSpeed

**Date:** 2026-06-17  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

PageSpeed Insights is introducing a new category for 'agentic browsing' audits, signaling a shift toward optimizing for AI-driven web agents.

## Details

The Categories schema now includes an 'agentic-browsing' property (type LighthouseCategoryV5). This addition allows the API to return a specific set of audits grouped under this new category, which is designed to measure how effectively a site can be navigated and understood by automated AI agents and LLM-based browsers.
