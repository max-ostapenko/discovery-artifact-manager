# Dataform adds managed GitRepositoryLink support

**Date:** 2026-06-14  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now allows linking Git repositories using managed credentials via the new gitRepositoryLink field in GitRemoteSettings.

## Details

The update introduces gitRepositoryLink to the GitRemoteSettings schema. This optional string field points to a GitRepositoryLink resource (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*). This enables developers to centralize Git credential management using Google Cloud's connection infrastructure instead of manually handling secrets or tokens within Dataform's Git settings.
