# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-14  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking to centralized Git repository connection resources for machine-based authentication.

## Details

The `GitRemoteSettings` schema has been updated with a new optional `gitRepositoryLink` field. This allows developers to reference a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`, facilitating the use of managed machine credentials instead of manual configuration for Git remote operations.
