# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-14  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's Git remote settings now support a gitRepositoryLink field for managing machine credentials via dedicated connection resources.

## Details

The GitRemoteSettings schema has been updated to include an optional gitRepositoryLink string field. This allows developers to point to a specific GitRepositoryLink resource (format: projects/*/locations/*/connections/*/gitRepositoryLinks/*) instead of relying on legacy credential methods, facilitating better integration with GCP's managed connection services.
