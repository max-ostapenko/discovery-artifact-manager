# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-14  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` in `GitRemoteSettings`, enabling managed connection resources for Git authentication in automated workflows.

## Details

The `GitRemoteSettings` schema has been updated with a new optional field: `gitRepositoryLink`. This field accepts a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`. This addition allows developers to leverage managed connectivity services (like Developer Connect) for machine-to-machine Git authentication, reducing the need for manual secret management in automated SQL workflows.
