# Dataform adds GitRepositoryLink for machine auth

**Date:** 2026-06-14  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` within `GitRemoteSettings`, enabling more streamlined machine credential management via Google Cloud connections.

## Details

The `GitRemoteSettings` schema has been updated with a new optional `gitRepositoryLink` string field. This field allows developers to reference a specific Git repository link resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) to handle machine credentials. This change likely integrates Dataform more deeply with Google Cloud's Developer Connect for managed repository authentication.
