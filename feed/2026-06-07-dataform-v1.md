# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-07  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform's GitRemoteSettings now includes a gitRepositoryLink field, enabling integration with managed Git repository connections.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field allows developers to specify a resource name (following the format projects/*/locations/*/connections/*/gitRepositoryLinks/*) to handle machine credentials. This suggests a move toward using Google Cloud's centralized connection management rather than manual credential handling within Dataform itself.
