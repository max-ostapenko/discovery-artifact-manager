# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-07  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's Git settings now support a gitRepositoryLink field, allowing developers to link to external Git repositories using standardized resource names.

## Details

The GitRemoteSettings schema has been updated with a new optional string field: gitRepositoryLink. This field expects a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*. This addition likely facilitates better integration with GCP's Developer Connect or similar connectivity services for managing machine-based Git authentication.
