# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-07  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform now supports linking to GitRepositoryLink resources for machine credentials in Git remote settings.

## Details

A new optional field `gitRepositoryLink` has been added to the `GitRemoteSettings` schema. This allows developers to specify a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*` to handle machine credentials for Git operations, streamlining how Dataform authenticates with remote repositories via Developer Connect.
