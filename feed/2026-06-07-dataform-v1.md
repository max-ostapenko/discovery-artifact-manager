# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-07  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports referencing GitRepositoryLink resources in Git remote settings, enabling centralized machine credential management.

## Details

A new optional field `gitRepositoryLink` has been added to the `GitRemoteSettings` schema. This allows developers to point to a managed Git repository connection resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) instead of configuring credentials manually within Dataform, likely integrating with GCP's Developer Connect service.
