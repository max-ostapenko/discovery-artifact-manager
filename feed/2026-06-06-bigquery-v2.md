# Iceberg Support and Python UDF Clarifications

**Date:** 2026-06-06  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery documentation updates reveal expanded support for Iceberg External Tables and clarify concurrency settings for Python UDFs.

## Details

The `timestampTargetPrecision` field in both `ExternalDataConfiguration` and `JobConfigurationLoad` now explicitly supports Iceberg External Tables for timestamp autodetection, joining CSV, PARQUET, and AVRO. Additionally, the `containerRequestConcurrency` field in `ExternalRuntimeOptions` has been clarified to apply specifically to Python UDF container instances rather than generic Cloud Run instances.
