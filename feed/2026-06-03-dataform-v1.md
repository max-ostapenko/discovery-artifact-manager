# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-03  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports referencing centralized GitRepositoryLink resources for machine-based authentication to remote repositories.

## Details

The GitRemoteSettings schema has been updated to include an optional gitRepositoryLink field. This field allows developers to provide a resource name (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*) to handle credentials, likely integrating with GCP's Developer Connect or similar connectivity services for more secure and centralized Git management.
