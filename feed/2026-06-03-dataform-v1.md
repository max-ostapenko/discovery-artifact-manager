# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-03  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now supports gitRepositoryLink, allowing developers to use managed connections for machine-based Git authentication.

## Details

The GitRemoteSettings schema has been updated with a new optional string field: gitRepositoryLink. This field accepts a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*, enabling tighter integration with GCP's managed connection services (Developer Connect) for repository access instead of relying solely on manual secret management.
