# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-03  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's Git settings now support a direct link to GitRepositoryLink resources, simplifying machine-based authentication.

## Details

The `GitRemoteSettings` schema has been updated with a new optional field: `gitRepositoryLink`. This field allows developers to specify a resource name (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) to handle machine credentials for Git operations, likely integrating with Google Cloud's managed connection services.
