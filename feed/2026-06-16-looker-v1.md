# Looker adds Instance Recovery and Release Channels

**Date:** 2026-06-16  
**API:** `looker.v1`  
**Impact:** Medium  

## Summary

Looker v1 now supports soft-delete recovery and configurable release channels, giving developers better control over instance lifecycle and update cadences.

## Details

A new `undelete` method has been added to the instances resource, allowing for the recovery of soft-deleted instances. To support this, the `list` method now includes a `showDeleted` parameter, and the Instance schema has been expanded with `softDeleteReason` and `suspendedTime` fields. Additionally, developers can now programmatically manage update frequency via the new `releaseChannel` field (supporting RAPID, REGULAR, and STABLE) and toggle `acceleratedSecurityPatchEnabled` for faster security updates.
