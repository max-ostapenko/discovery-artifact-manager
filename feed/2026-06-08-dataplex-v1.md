# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-08  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its DataScan capabilities with support for unstructured data profiling and semantic graph profiling.

## Details

Dataplex has introduced a new scan type, `UNSTRUCTURED_DATA_PROFILE`, allowing developers to profile non-tabular data sources. This includes new specification and result fields in the `DataScan` and `DataScanJob` resources. Additionally, a large set of schemas for `GraphProfile` has been added, enabling semantic inference and relationship mapping through node and edge type definitions. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to prevent truncation of schema fields in the output context.
