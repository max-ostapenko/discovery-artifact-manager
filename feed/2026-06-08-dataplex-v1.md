# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-08  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its data governance capabilities with support for unstructured data profile scans and new graph-based semantic inference profiling.

## Details

This update introduces the `UNSTRUCTURED_DATA_PROFILE` scan type to `DataScan` and `DataScanJob` resources, accompanied by new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. Additionally, a comprehensive set of schemas for Graph Profiling has been added (including `GraphProfileEdgeType` and `GraphProfileNodeType`), enabling developers to analyze semantic relationships and connectivity within their data. A minor update to `LookupContextRequest` also adds the `all_schema_fields` option to prevent truncation of schema fields in the output context.
