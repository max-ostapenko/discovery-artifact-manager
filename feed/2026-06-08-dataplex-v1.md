# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-08  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new unstructured data profiling type and a comprehensive graph-based semantic inference model.

## Details

The DataScan and DataScanJob resources now support a new scan type: `UNSTRUCTURED_DATA_PROFILE`. This includes the addition of `unstructuredDataProfileSpec` for configuration and `unstructuredDataProfileResult` for output. Furthermore, a significant set of schemas for graph profiling has been introduced (e.g., `GraphProfile`, `GraphProfileEdgeType`, `GraphProfileNodeType`), allowing developers to model and retrieve semantic relationships, edge cardinalities, and foreign key constraints. A minor update to `LookupContextRequest` also adds the `all_schema_fields` option to prevent truncation of schema metadata in responses.
