# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-08  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new scan type for unstructured data and a comprehensive GraphProfile schema for semantic inference.

## Details

The DataScan and DataScanJob resources now support a new enum value, UNSTRUCTURED_DATA_PROFILE, allowing developers to configure and retrieve results for unstructured data scans via the new unstructuredDataProfileSpec and unstructuredDataProfileResult fields. Additionally, a large suite of schemas has been added to support GraphProfile, which includes detailed structures for node types, edge types, and foreign key constraints, likely powering new semantic inference features. Finally, the LookupContextRequest now includes an all_schema_fields option to bypass character budget limits and return full schema metadata.
