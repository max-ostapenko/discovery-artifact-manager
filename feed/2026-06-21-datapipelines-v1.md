# Documentation Update for Pipeline Sources

**Date:** 2026-06-21  
**API:** `datapipelines.v1`  
**Impact:** Low  

## Summary

A minor documentation update to the Data Pipelines API clarifies pipeline source examples.

## Details

The description for the `pipelineSources` field in the `Pipeline` schema has been updated. It now references 'Knowledge Catalog' instead of 'Dataplex' as an example of a source that sets keys and values during pipeline creation. This is a non-functional change.
