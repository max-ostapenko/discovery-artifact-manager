# Documentation update for pipeline sources

**Date:** 2026-06-21  
**API:** `datapipelines.v1`  
**Impact:** Low  

## Summary

A minor documentation update to the Data Pipelines API clarifies pipeline source examples, replacing Dataplex with Knowledge Catalog.

## Details

The description for the `pipelineSources` field within the `GoogleCloudDatapipelinesV1Pipeline` schema has been updated. The example source provided in the documentation was changed from 'Dataplex' to 'Knowledge Catalog'. This change is purely descriptive and does not affect API functionality or schema structure.
