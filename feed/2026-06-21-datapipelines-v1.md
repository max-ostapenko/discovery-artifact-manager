# Minor Documentation Update for Pipeline Sources

**Date:** 2026-06-21  
**API:** `datapipelines.v1`  
**Impact:** Low  

## Summary

The description for pipeline sources has been updated to reference Knowledge Catalog instead of Dataplex as an example.

## Details

The pipelineSources field in the GoogleCloudDatapipelinesV1Pipeline schema received a minor description update. The example provided in the documentation was changed from 'Dataplex' to 'Knowledge Catalog'. This is a non-functional change and does not affect API behavior or schema structure.
