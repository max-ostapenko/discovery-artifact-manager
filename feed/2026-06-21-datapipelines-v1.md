# Documentation update for pipeline sources

**Date:** 2026-06-21  
**API:** `datapipelines.v1`  
**Impact:** Low  

## Summary

The description for pipeline sources has been updated to reference Knowledge Catalog instead of Dataplex as an example.

## Details

This is a documentation-only change to the pipelineSources field within the GoogleCloudDatapipelinesV1Pipeline schema. The example provided in the description was updated from 'Dataplex' to 'Knowledge Catalog'. There are no functional changes to the API or its behavior.
