# Minor documentation cleanup and formatting

**Date:** 2026-06-24  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

This update consists of minor documentation refinements, including the removal of internal linting comments and improved formatting for resource names in descriptions.

## Details

The changes are limited to description updates across several resources, including CompilationResults, ReleaseConfigs, and GitRemoteSettings. Specifically, internal 'LINT.ThenChange' markers were removed from multiple parameter and schema descriptions, and backticks were added to resource names like 'TeamFolders' and 'GitRepositoryLink' for better readability in the documentation. No functional changes or schema modifications were introduced.
