# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-06  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` in Git remote settings, enabling easier integration with managed Git connections for machine credentials.

## Details

The `GitRemoteSettings` resource has a new optional field `gitRepositoryLink`. This field accepts a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`, allowing Dataform to leverage centralized Git repository links for authentication rather than relying solely on manual credential configurations.
