# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-06  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform now supports referencing GitRepositoryLink resources for machine-based authentication, streamlining how Git connections are managed.

## Details

The GitRemoteSettings schema now includes an optional gitRepositoryLink field. This allows developers to point to a specific connection resource (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*) for machine credentials. This update likely integrates Dataform with Google Cloud's managed connection services, reducing the need for manual secret handling for Git operations.
