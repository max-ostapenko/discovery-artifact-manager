# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-06  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform's Git remote settings now support gitRepositoryLink, allowing developers to link to centralized Git repository connection resources for machine-based authentication.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field accepts a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*. This change facilitates better credential management by leveraging external connection resources instead of relying solely on inline or secret-based configurations.
