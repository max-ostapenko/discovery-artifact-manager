# Deprecation of Destinations get and link methods

**Date:** 2026-06-21  
**API:** `tagmanager.v2`  
**Impact:** Medium  

## Summary

Google Tag Manager has officially marked the 'get' and 'link' methods within the destinations resource as deprecated.

## Details

The API discovery document now explicitly flags accounts.containers.destinations.get and accounts.containers.destinations.link with a deprecated status. This indicates that these specific endpoints for managing container destinations are on the path to retirement, and developers should look for alternative integration patterns for tagging configurations.
