# GTM Destination methods marked as deprecated

**Date:** 2026-06-21  
**API:** `tagmanager.v2`  
**Impact:** Medium  

## Summary

The get and link methods for Tag Manager Destinations are now officially deprecated, signaling a shift in how container destinations are managed.

## Details

Two specific methods within the Tag Manager v2 API have been flagged as deprecated: `accounts.containers.destinations.get` and `accounts.containers.destinations.link`. While these methods remain functional for now, their deprecation indicates that the underlying resource model for destinations is likely being phased out or replaced by a different configuration workflow.
