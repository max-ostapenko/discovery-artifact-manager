# Deprecation of Destination methods in Tag Manager

**Date:** 2026-06-21  
**API:** `tagmanager.v2`  
**Impact:** Medium  

## Summary

Google Tag Manager has officially marked the 'get' and 'link' methods for the destinations resource as deprecated.

## Details

The Tag Manager API has flagged two methods within the destinations resource as deprecated: accounts.containers.destinations.get and accounts.containers.destinations.link. While these methods are still currently available, the deprecation status indicates that developers should begin looking for alternative ways to manage container destinations as these endpoints may be removed in a future revision.
