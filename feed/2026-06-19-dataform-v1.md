# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-19  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` within `GitRemoteSettings`, enabling centralized machine credential management for Git operations.

## Details

The `GitRemoteSettings` schema has been updated with a new optional field: `gitRepositoryLink`. This field accepts a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`. This addition allows Dataform to leverage Google Cloud's managed Git repository connections, facilitating more secure and centralized machine credential handling for repository synchronization.
