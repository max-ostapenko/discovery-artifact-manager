# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-19  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking to GitRepositoryLink resources for machine-based authentication in Git remote settings.

## Details

A new optional field `gitRepositoryLink` has been added to the `GitRemoteSettings` schema. This field allows developers to specify a resource name (following the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) to manage machine credentials, likely leveraging Google Cloud's standardized repository connection services instead of manual token or SSH key management.
