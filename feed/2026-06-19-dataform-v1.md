# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-19  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's Git settings now support a gitRepositoryLink field, enabling the use of managed machine credentials via standard GCP connection resources.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field accepts a resource path (projects/*/locations/*/connections/*/gitRepositoryLinks/*), allowing developers to link Dataform workspaces to Git repositories using centralized credential management rather than manual secret handling.
