# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-19  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking Git repositories via managed connection resources for more secure machine credential handling.

## Details

The GitRemoteSettings schema has been updated to include a new optional field, gitRepositoryLink. This field allows developers to specify a resource name (following the format projects/*/locations/*/connections/*/gitRepositoryLinks/*) to handle machine credentials. This integration suggests a move toward using Google Cloud's centralized connection management for Git operations instead of manual credential configuration.
