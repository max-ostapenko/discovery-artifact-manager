# Deprecation of Container Destination methods

**Date:** 2026-06-12  
**API:** `tagmanager.v2`  
**Impact:** Medium  

## Summary

Google Tag Manager has officially marked the 'get' and 'link' methods for container destinations as deprecated.

## Details

The API discovery document now flags accounts.containers.destinations.get and accounts.containers.destinations.link with a deprecated status. While these methods remain functional for now, their inclusion in the deprecation cycle suggests a shift in how GTM handles destination resources, and developers should prepare to migrate away from these specific endpoints.
