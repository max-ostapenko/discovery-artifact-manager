# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-14  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex DataScan now supports unstructured data profiling and introduces a comprehensive schema for graph-based semantic inference.

## Details

This update introduces the `UNSTRUCTURED_DATA_PROFILE` scan type to `GoogleCloudDataplexV1DataScan`, allowing developers to profile non-tabular data via the new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. Additionally, a large suite of schemas for `GraphProfile` has been added, enabling the representation of complex relationships through `nodeTypes`, `edgeTypes`, and `foreignKeys`. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to ensure full schema visibility regardless of the character budget.
