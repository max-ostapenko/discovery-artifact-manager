# Dataplex adds Unstructured Data Profiling and Graph Discovery

**Date:** 2026-06-14  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex V1 now supports profiling for unstructured data and introduces a comprehensive Graph Profile schema for semantic inference and relationship mapping.

## Details

The API has introduced `UNSTRUCTURED_DATA_PROFILE` as a new scan type within `DataScan` and `DataScanJob` resources, supported by the new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. Additionally, a massive suite of Graph Profile schemas (including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`) has been added to support semantic inference, allowing developers to model and discover data relationships via nodes, edges, and foreign key constraints. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to return full schema details regardless of the character budget.
