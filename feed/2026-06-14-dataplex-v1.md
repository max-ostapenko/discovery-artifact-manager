# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-14  
**API:** `dataplex.v1`  
**Impact:** High  

## Summary

Dataplex expands its scanning capabilities with a new unstructured data profile scan type and comprehensive support for graph-based semantic inference results.

## Details

The DataScan and DataScanJob resources now support a new enum value, UNSTRUCTURED_DATA_PROFILE, along with associated spec and result schemas (unstructuredDataProfileSpec and unstructuredDataProfileResult). This update also introduces a significant suite of schemas for GraphProfile, including EdgeTypes and NodeTypes, which allow for detailed mapping of relationships, foreign keys, and semantic structures within data scans. Additionally, the LookupContextRequest now supports an all_schema_fields option to bypass context budget limits for schema field retrieval.
