# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-14  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex is expanding its scanning capabilities with support for unstructured data profiling and a new graph-based semantic inference engine.

## Details

The DataScan API now supports a new `UNSTRUCTURED_DATA_PROFILE` type, accompanied by `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields for analyzing non-tabular data. Additionally, a comprehensive suite of Graph Profile schemas has been introduced (including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`), enabling the service to map semantic relationships, nodes, and edges within data lakes. Finally, `LookupContextRequest` now includes an `all_schema_fields` option to bypass context budget truncation and return full schema details.
