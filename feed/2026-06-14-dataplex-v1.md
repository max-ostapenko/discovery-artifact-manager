# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-14  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex is expanding its scanning capabilities to include unstructured data profiling and semantic graph inference for complex data relationships.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE type, accompanied by specific spec and result schemas. Additionally, a large set of schemas for GraphProfile has been introduced, allowing Dataplex to model and infer graph structures (nodes and edges) from data, including support for foreign keys, cardinality hints, and field mappings. Finally, LookupContextRequest now supports an all_schema_fields option to bypass context budget limits and return exhaustive schema metadata.
