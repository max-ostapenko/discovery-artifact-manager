# Documentation update for pipeline sources

**Date:** 2026-06-20  
**API:** `datapipelines.v1`  
**Impact:** Low  

## Summary

A minor documentation update to the Data Pipelines API clarifies pipeline source examples, replacing Dataplex with Knowledge Catalog.

## Details

The description for the `pipelineSources` field in the `GoogleCloudDatapipelinesV1Pipeline` schema has been updated. The change replaces the example source 'Dataplex' with 'Knowledge Catalog'. This is a purely descriptive change and does not affect API functionality or schema structure.
