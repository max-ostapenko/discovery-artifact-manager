# Iceberg Support for Timestamp Autodetection

**Date:** 2026-06-02  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery's timestamp autodetection now supports Iceberg External Tables, and Python UDF concurrency settings have been clarified.

## Details

The timestampTargetPrecision field in both ExternalDataConfiguration and JobConfigurationLoad now explicitly supports Iceberg External Tables alongside CSV, Parquet, and Avro. This allows for more granular control over timestamp precision during schema autodetection for Iceberg sources. Additionally, the description for containerRequestConcurrency in ExternalRuntimeOptions was updated to specify it applies to Python UDF container instances, providing better clarity for developers tuning user-defined function performance.
