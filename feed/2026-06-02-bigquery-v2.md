# Iceberg support and Python UDF documentation updates

**Date:** 2026-06-02  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery documentation now explicitly includes support for Iceberg External Tables in timestamp precision settings and clarifies concurrency for Python UDFs.

## Details

The `timestampTargetPrecision` field within `ExternalDataConfiguration` and `JobConfigurationLoad` has been updated to include 'Iceberg External Table' as a supported format for timestamp autodetection. Additionally, the description for `containerRequestConcurrency` in `ExternalRuntimeOptions` was refined to specify its role in Python UDF container instances, moving away from a generic Cloud Run description.
