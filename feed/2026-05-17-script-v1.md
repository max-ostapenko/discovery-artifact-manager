# Massive removal of Apps Script API v1 discovery metadata

**Date:** 2026-05-17  
**API:** `script.v1`  
**Impact:** High  
**⚠️ Breaking change**  

## Summary

The discovery document for Apps Script v1 has been stripped of nearly all its metadata, including OAuth2 scopes and global parameters.

## Details

Over 800 entries were removed from the discovery document. This includes the removal of critical OAuth2 scopes for Gmail, Drive, Calendar, and Sheets, as well as the management scopes for script deployments, metrics, and projects. Global parameters such as access_token, fields, and prettyPrint were also removed. This suggests a major deprecation or a fundamental shift in how this API surface is exposed to client library generators.
