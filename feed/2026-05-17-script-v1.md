# Apps Script API v1 discovery metadata removed

**Date:** 2026-05-17  
**API:** `script.v1`  
**Impact:** High  
**⚠️ Breaking change**  

## Summary

The Apps Script API v1 discovery document has been effectively gutted, removing core metadata, global parameters, and OAuth2 scope definitions.

## Details

A massive update has removed 868 elements from the discovery document. This includes top-level API metadata (name, canonicalName, description), all standard global parameters (access_token, alt, fields, key, prettyPrint), and descriptions for nearly 20 OAuth2 scopes including Gmail, Drive, Calendar, and Sheets. Such a comprehensive removal of metadata typically signals that this version of the discovery document is being decommissioned or the API surface is being hidden from public discovery tools.
