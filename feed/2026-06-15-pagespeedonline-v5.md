# New 'Agentic Browsing' category added to PageSpeed audits

**Date:** 2026-06-15  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now supports a new 'agentic-browsing' category in Lighthouse reports, focusing on audits for automated agents.

## Details

A new field `agentic-browsing` has been added to the `Categories` schema. It references the `LighthouseCategoryV5` type and is designed to house audits related to agentic browsing. This addition suggests Google is beginning to formalize how websites are measured for compatibility with AI agents and autonomous browsers, moving beyond traditional human-centric performance metrics.
