# New 'Agentic Browsing' Category in Lighthouse Reports

**Date:** 2026-06-15  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

PageSpeed Insights has introduced a new 'agentic-browsing' category to the Lighthouse Categories schema, targeting automated or AI-driven browsing audits.

## Details

The API's Categories schema now includes a property for 'agentic-browsing', which references the LighthouseCategoryV5 object. This addition suggests a new set of audits focused on how AI agents or automated browsers interact with web content, providing developers with a specific category to track performance and accessibility for non-human visitors.
