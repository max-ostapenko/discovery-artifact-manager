# Dataplex adds Change Request workflow and removes GenAI field

**Date:** 2026-05-22  
**API:** `dataplex.v1`  
**Impact:** Medium  
**⚠️ Breaking change**  

## Summary

Dataplex introduces a new ChangeRequest resource for formalizing resource modifications and removes a deprecated GenAI inference field.

## Details

A new resource hierarchy, `projects.locations.changeRequests`, has been added to the API. This includes methods for `list`, `get`, `delete`, and a specialized `approve` action, enabling programmatic governance workflows for data management. On the discovery side, the `entityInferenceEnabled` field has been removed from `GoogleCloudDataplexV1DataDiscoverySpecStorageConfigUnstructuredDataOptions`. Developers who were using this deprecated field for GenAI-based entity inference must migrate to `semantic_inference_enabled`.
