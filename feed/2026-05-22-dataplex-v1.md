# Dataplex adds Change Requests for governed resource updates

**Date:** 2026-05-22  
**API:** `dataplex.v1`  
**Impact:** Medium  
**⚠️ Breaking change**  

## Summary

Dataplex introduces a formal Change Request workflow, allowing developers to propose, list, and approve changes to data resources.

## Details

A new resource hierarchy `projects.locations.changeRequests` has been added. It includes methods for `approve`, `get`, `list`, and `delete`. Notably, the `delete` method implements state-based permissions: standard users can only delete requests in the `NEW` state, whereas admins have full deletion rights. Additionally, the deprecated `entityInferenceEnabled` field was removed from the `DataDiscoverySpec` for unstructured data, finalizing its transition to semantic inference.
