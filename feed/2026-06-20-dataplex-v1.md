# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-20  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its DataScan capabilities with support for unstructured data profiling and a new graph-based profiling schema for semantic inference.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE type, accompanied by specific Spec and Result schemas for managing these scans. Additionally, a comprehensive suite of GraphProfile schemas has been introduced (including NodeTypes, EdgeTypes, and ForeignKeys), enabling structured semantic inference and relationship mapping within data scans. Finally, the LookupContextRequest now includes an all_schema_fields option to prevent the truncation of schema fields in the output context.
