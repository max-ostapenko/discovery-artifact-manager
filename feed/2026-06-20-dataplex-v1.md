# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-20  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new scan type for unstructured data and a comprehensive Graph Profile schema for semantic inference.

## Details

Dataplex has introduced the `UNSTRUCTURED_DATA_PROFILE` scan type to `DataScan` and `DataScanJob` resources, supported by new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. Additionally, a significant new Graph Profile feature has been added, including schemas for `GraphProfile`, `GraphProfileNodeType`, and `GraphProfileEdgeType`. This allows the API to represent complex semantic relationships, including edge cardinality, node types, and foreign key constraints. Finally, `LookupContextRequest` now includes an `all_schema_fields` option to bypass character-budget truncation when retrieving schema metadata.
