# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-20  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex is expanding its scanning capabilities with a new unstructured data profile scan type and a comprehensive graph profiling schema for semantic inference.

## Details

This update introduces the `UNSTRUCTURED_DATA_PROFILE` scan type to both `DataScan` and `DataScanJob` resources, supported by new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. Additionally, a large set of schemas for Graph Profiling has been added (e.g., `GoogleCloudDataplexV1GraphProfile`), allowing developers to model and retrieve semantic relationships between data entities via node and edge types, including foreign key constraints and cardinality hints. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to bypass character-budget truncation when retrieving schema metadata.
