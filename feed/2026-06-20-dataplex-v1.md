# Dataplex adds unstructured data profiling and graph mapping

**Date:** 2026-06-20  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex now supports profiling unstructured data and inferring semantic relationships via new Graph Profile schemas and DataScan types.

## Details

The DataScan and DataScanJob resources have been expanded with a new UNSTRUCTURED_DATA_PROFILE type, including specific specification and result schemas (unstructuredDataProfileSpec/Result). A significant set of new schemas for Graph Profiling has also been added, including GraphProfile, EdgeType, and NodeType, enabling semantic inference scans that map relationships between data entities. Additionally, LookupContextRequest now supports an all_schema_fields boolean to bypass context budget truncation for schema fields.
