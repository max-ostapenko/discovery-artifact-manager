# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-20  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex introduces support for unstructured data profile scans and a comprehensive graph profiling system for semantic inference.

## Details

The DataScan resource now supports a new UNSTRUCTURED_DATA_PROFILE type, accompanied by specific spec and result schemas for profiling non-tabular data. Additionally, a large suite of Graph Profile schemas has been added (including GraphProfile, GraphProfileEdgeType, and GraphProfileNodeType), enabling the discovery and modeling of complex data relationships, edge cardinalities, and foreign key constraints. The LookupContextRequest also gains an all_schema_fields option to bypass character budget limits when retrieving schema metadata.
