# Dataform Git settings and directory path improvements

**Date:** 2026-05-20  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform adds an output-only field for effective Git branches and clarifies directory path structures.

## Details

The GitRemoteSettings schema now includes `effectiveDefaultBranch`, an output-only field that reveals the actual branch being used (defaulting to 'main' if not specified). Additionally, the `defaultBranch` field has been updated from required to optional. For developers working with file structures, `DirectoryEntry` descriptions now explicitly state that file and directory paths include the full folder structure from the root. Finally, the search method for team folders now documents a default page size of 50.
