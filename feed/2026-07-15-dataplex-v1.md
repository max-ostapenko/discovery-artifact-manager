# Breaking: context field removed from LookupContextRequest

**Date:** 2026-07-15  
**API:** `dataplex.v1`  
**Impact:** Medium  
**⚠️ Breaking change**  

## Summary

The optional context string has been removed from the LookupContextRequest schema, affecting metadata lookup workflows.

## Details

The GoogleCloudDataplexV1LookupContextRequest schema has been modified to remove the context property. This field was previously used to provide optional textual contextual information for metadata context requests. Developers using this specific lookup functionality should update their request payloads to remove this field, as it is no longer recognized by the API schema.
