# Vertex AI adds Exa.ai grounding and Veo video experiments

**Date:** 2026-06-15  
**API:** `aiplatform.v1`  
**Impact:** Medium  

## Summary

Vertex AI expands its generative capabilities with Exa.ai search grounding, advanced video generation controls like seamless looping, and enhanced Computer Use safety features.

## Details

This update introduces several experimental features for video generation (Veo), including 'seamless' parameters for temporal looping and spatial tessellation, and support for the DNxHR codec. Developers can now use the 'exaAiSearch' tool for grounding model responses via Exa.ai. The 'Computer Use' tool gains a prompt injection detection toggle and specific environment flags for mobile and desktop. On the infrastructure side, PersistentDiskSpec now supports various Hyperdisk types (ML, Extreme, Throughput) for high-performance workloads. Note that Google Maps contextual widgets are now deprecated in grounding responses.
