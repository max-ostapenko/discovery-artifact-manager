# Vertex AI adds Veo video features and Exa.ai grounding

**Date:** 2026-06-15  
**API:** `aiplatform.v1`  
**Impact:** Medium  

## Summary

Vertex AI introduced advanced video generation controls for Veo, integrated Exa.ai for search grounding, and added prompt injection detection for computer-use tools.

## Details

The video generation suite (Veo) receives significant experimental updates, including 'seamless' parameters for temporal looping and spatial tessellation, plus support for the professional VIDEO_CODEC_DNXHR codec. Grounding capabilities are expanded with the new 'exaAiSearch' tool, allowing models to use Exa.ai for real-time information retrieval. For agentic workflows, 'ToolComputerUse' now supports prompt injection detection and environment-specific flags for mobile and desktop. Infrastructure-wise, 'PersistentDiskSpec' now supports several Hyperdisk types (e.g., hyperdisk-ml, hyperdisk-balanced) for high-performance storage requirements.
