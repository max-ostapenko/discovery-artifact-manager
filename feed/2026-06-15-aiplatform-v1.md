# Vertex AI adds Veo Video Looping and Exa.ai Grounding

**Date:** 2026-06-15  
**API:** `aiplatform.v1`  
**Impact:** Medium  

## Summary

Vertex AI introduces advanced video generation controls for Veo, Exa.ai search grounding, and expanded 'Computer Use' safety features.

## Details

The API adds significant capabilities to video generation via CloudAiLargeModelsVisionGenerateVideoExperiments, including 'seamless' parameters for temporal looping and spatial tessellation, and support for the VIDEO_CODEC_DNXHR codec. A new 'exaAiSearch' tool enables grounding model responses using Exa.ai. For agentic workflows, ToolComputerUse now includes prompt injection detection and environment flags for mobile and desktop. Additionally, PersistentDiskSpec now supports various Hyperdisk types (ML, Balanced, Throughput), and Google Maps contextual widgets are officially deprecated.
