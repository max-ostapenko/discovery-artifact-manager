# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-05  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now includes a gitRepositoryLink field, enabling the use of managed Git connections for machine-based authentication.

## Details

The addition of `gitRepositoryLink` to the `GitRemoteSettings` schema allows developers to reference a specific GitRepositoryLink resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`). This provides a more structured and potentially more secure way to handle Git credentials compared to manual secret management, aligning Dataform with GCP's broader connection management infrastructure.
