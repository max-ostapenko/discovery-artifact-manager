# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-05  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking to centralized Git repository resources for machine credentials, improving how SQL workflows connect to remote repositories.

## Details

A new `gitRepositoryLink` field has been added to the `GitRemoteSettings` schema. This allows developers to reference a specific `GitRepositoryLink` resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) instead of relying solely on traditional credential methods. This aligns Dataform with GCP's broader Developer Connect infrastructure for more secure and manageable Git integrations.
