# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-05  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports managed Git repository links, simplifying how machine credentials are handled for remote Git repositories.

## Details

The `GitRemoteSettings` resource has been updated with a new optional field: `gitRepositoryLink`. This field allows developers to reference a managed repository connection resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) for authentication. This shift suggests better integration with Google Cloud's centralized repository management, reducing the need to manually handle secrets or SSH keys for machine-to-machine Git operations.
