# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-05  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports referencing managed Git repository links within GitRemoteSettings, streamlining machine-based authentication for SQL workflows.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field accepts a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*. This addition allows developers to leverage Google Cloud's native connection management for Git credentials, providing a more integrated way to handle machine-based authentication for repository syncing.
