# New 'Agentic Browsing' category added to Lighthouse audits

**Date:** 2026-06-14  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now includes an 'agentic-browsing' category in its Lighthouse results, enabling developers to track audits specifically for automated or AI-driven browsing.

## Details

A new property `agentic-browsing` has been added to the `Categories` schema, referencing the `LighthouseCategoryV5` type. This addition suggests Google is beginning to surface metrics or audits related to how web pages interact with agentic AI or automated browsing systems, allowing developers to optimize for non-human visitors.
