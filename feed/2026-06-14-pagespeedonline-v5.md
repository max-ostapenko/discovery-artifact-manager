# PageSpeed Insights adds Agentic Browsing category

**Date:** 2026-06-14  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

The API now includes a new 'agentic-browsing' category in Lighthouse reports, signaling a shift toward measuring how AI agents perceive web content.

## Details

The Categories schema has been updated to include a new property: agentic-browsing. This field uses the LighthouseCategoryV5 schema and is intended to group audits that evaluate how well a page performs when accessed by automated or AI-driven browsing agents.
