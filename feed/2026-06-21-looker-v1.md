# Looker v1 adds Instance Undelete and Release Channels

**Date:** 2026-06-21  
**API:** `looker.v1`  
**Impact:** Medium  

## Summary

Looker Core now supports soft-delete recovery and programmatic control over update release channels.

## Details

A new `instances.undelete` method has been introduced, allowing developers to restore instances that were previously deleted. To support this, the `instances.list` method now accepts a `showDeleted` boolean parameter. The `Instance` schema has also been expanded with `releaseChannel` (offering RAPID, REGULAR, and STABLE tracks), `acceleratedSecurityPatchEnabled` for faster security updates, and metadata fields like `softDeleteReason` and `suspendedTime` to help diagnose why an instance was taken offline.
