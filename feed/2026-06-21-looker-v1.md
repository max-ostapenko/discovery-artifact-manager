# Looker Core adds Soft Delete recovery and Release Channels

**Date:** 2026-06-21  
**API:** `looker.v1`  
**Impact:** Medium  

## Summary

Looker v1 now supports instance recovery via a new undelete method and provides granular control over update cycles with release channel settings.

## Details

The instances.list method now includes a showDeleted parameter to view instances in a soft-deleted state. A new undelete method has been added to restore these instances. Additionally, the Instance schema now supports releaseChannel (RAPID, REGULAR, STABLE) for managing update frequency and acceleratedSecurityPatchEnabled for opting into faster security updates. New output-only fields like softDeleteReason and suspendedTime provide better visibility into instance lifecycle events.
