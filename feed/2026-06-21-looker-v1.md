# Looker Core adds Soft Delete recovery and Release Channels

**Date:** 2026-06-21  
**API:** `looker.v1`  
**Impact:** Medium  

## Summary

Looker v1 now supports instance recovery via a new undelete method and provides controls for update release channels and accelerated security patches.

## Details

The API introduces a 'soft delete' lifecycle management suite. Developers can now use the new `instances.undelete` method to restore instances, and the `instances.list` method now accepts a `showDeleted` parameter. The `Instance` schema has been expanded with `softDeleteReason`, `suspendedTime`, and a `releaseChannel` enum (RAPID, REGULAR, STABLE) to control update cadences. Additionally, a new `acceleratedSecurityPatchEnabled` boolean allows for faster security updates.
