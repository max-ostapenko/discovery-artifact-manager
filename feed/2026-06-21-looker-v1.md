# Looker v1 adds Instance Undelete and Release Channels

**Date:** 2026-06-21  
**API:** `looker.v1`  
**Impact:** Medium  

## Summary

Looker Core instances now support soft-deletion recovery and configurable update release channels (Rapid, Regular, Stable).

## Details

This update introduces a significant lifecycle management feature: the `undelete` method for Looker instances. To support this, the `list` method now accepts a `showDeleted` parameter, and the `Instance` schema includes new fields like `softDeleteReason` and `suspendedTime`. Additionally, developers can now manage update cadences via the `releaseChannel` field and opt-in to `acceleratedSecurityPatchEnabled` for faster security updates.
