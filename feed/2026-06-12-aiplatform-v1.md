# Vertex AI adds Veo video experiments and Exa.ai grounding

**Date:** 2026-06-12  
**API:** `aiplatform.v1`  
**Impact:** Medium  

## Summary

Vertex AI introduces advanced video generation controls for Veo, a new Exa.ai search tool for model grounding, and expanded Hyperdisk support for machine specs.

## Details

The video generation suite (Veo) receives significant experimental updates, including support for seamless temporal looping, spatial tessellation (horizontal and vertical), and the DNxHR codec. A new grounding tool, `GoogleCloudAiplatformV1ToolExaAiSearch`, has been added, allowing models to query Exa.ai directly using a developer-provided API key. Additionally, `PersistentDiskSpec` now supports several Hyperdisk variants, including `hyperdisk-ml` and `hyperdisk-throughput`, and documentation now reflects a shift in terminology from 'Vertex AI' to 'Agent Platform' for resource management.
