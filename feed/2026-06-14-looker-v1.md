# Looker adds Soft Delete recovery and Release Channels

**Date:** 2026-06-14  
**API:** `looker.v1`  
**Impact:** Medium  

## Summary

Looker instances now support a soft-delete lifecycle with a new recovery method and configurable update release channels.

## Details

The API introduces a new `instances.undelete` method and a `showDeleted` parameter for `instances.list`, allowing developers to recover and track instances deleted due to billing issues, trial expirations, or manual requests. Additionally, the `Instance` schema now includes `releaseChannel` (Rapid, Regular, Stable) for controlling update frequency and `acceleratedSecurityPatchEnabled` for opting into faster security rollouts.
