# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-15  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new scan type for unstructured data and introduces comprehensive schemas for graph-based semantic inference.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE type, accompanied by specific spec and result fields (unstructuredDataProfileSpec and unstructuredDataProfileResult). Additionally, a significant set of schemas for GraphProfile has been introduced, covering node types, edge types, and foreign key constraints, which suggests new capabilities for graph-based data modeling and semantic inference. Developers using LookupContextRequest can also now use the all_schema_fields option to ensure all fields are returned regardless of the context budget.
