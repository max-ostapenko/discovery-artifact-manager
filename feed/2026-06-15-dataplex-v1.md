# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-15  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new scan type for unstructured data and a comprehensive graph-based semantic inference model.

## Details

The DataScan and DataScanJob resources now support a new `UNSTRUCTURED_DATA_PROFILE` enum, accompanied by `unstructuredDataProfileSpec` for configuration and `unstructuredDataProfileResult` for output. Additionally, a large suite of schemas has been introduced for `GraphProfile`, enabling the representation of data as nodes and edges to support semantic inference, including edge types, node types, and foreign key constraints. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to bypass truncation when retrieving metadata.
