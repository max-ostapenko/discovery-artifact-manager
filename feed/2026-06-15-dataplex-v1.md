# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-15  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex is expanding its scanning capabilities with a new unstructured data profiling type and a comprehensive graph-based semantic inference system.

## Details

The DataScan and DataScanJob resources now support a new `UNSTRUCTURED_DATA_PROFILE` enum, accompanied by specific `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields for managing scans on non-tabular data. Additionally, a massive set of schemas for Graph Profiling has been introduced, including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`. This allows the API to represent complex semantic relationships, cardinality (e.g., 1:N), and foreign key constraints discovered during scans. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to prevent truncation of schema metadata.
