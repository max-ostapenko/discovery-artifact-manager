# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-15  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex introduces support for profiling unstructured data and a comprehensive new Graph Profile schema for semantic inference and relationship mapping.

## Details

This update adds a new `UNSTRUCTURED_DATA_PROFILE` scan type to `DataScan` and `DataScanJob` resources, enabling automated profiling of non-tabular data. A significant set of new schemas for `GraphProfile` has been introduced, including `GraphProfileEdgeType` and `GraphProfileNodeType`, which allow for detailed semantic inference results including node/edge relationships and foreign key constraints. Additionally, the `LookupContextRequest` now includes an `all_schema_fields` option to force the return of all schema fields regardless of the character budget.
