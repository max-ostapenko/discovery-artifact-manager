# Dataform adds managed GitRepositoryLink support

**Date:** 2026-06-16  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports referencing managed Git repository connections for machine credentials via a new gitRepositoryLink field.

## Details

The `GitRemoteSettings` schema has been expanded to include `gitRepositoryLink`. This optional string field allows developers to point to a specific connection resource (format: `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) rather than relying on manual credential management. This update likely facilitates better integration with GCP's managed developer connection services for automated SQL workflows.
