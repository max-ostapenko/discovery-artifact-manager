# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-16  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now includes a gitRepositoryLink field, allowing for more robust machine credential management via resource names.

## Details

The update introduces the gitRepositoryLink property to the GitRemoteSettings schema. This optional string field allows developers to reference a specific GitRepositoryLink resource (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*). This move signals better integration with GCP's managed connection services, facilitating more secure and automated Git authentication for Dataform workspaces.
