# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-16  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking Git remote settings to managed GitRepositoryLink resources, enabling better machine-to-machine authentication.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field accepts a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*. This allows Dataform to leverage centralized connection management for Git operations, moving away from manual credential handling for automated workflows.
