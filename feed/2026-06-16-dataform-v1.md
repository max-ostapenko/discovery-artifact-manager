# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-16  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking to centralized Git repository connections for machine-based authentication via a new gitRepositoryLink field.

## Details

The GitRemoteSettings schema has been updated to include an optional gitRepositoryLink field. This field accepts a resource name following the pattern projects/*/locations/*/connections/*/gitRepositoryLinks/*, enabling developers to leverage Google Cloud's Developer Connect for managing machine credentials instead of manual configuration.
