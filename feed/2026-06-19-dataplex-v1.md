# Dataplex adds Unstructured Data Profiling and Graph Metadata

**Date:** 2026-06-19  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its DataScan capabilities with support for unstructured data profiling and a new graph-based metadata structure for semantic inference.

## Details

Dataplex is moving beyond structured data with the introduction of the `UNSTRUCTURED_DATA_PROFILE` scan type in `DataScan` and `DataScanJob`. This includes new specification and result schemas (`unstructuredDataProfileSpec`, `unstructuredDataProfileResult`). Additionally, a massive set of schemas for `GraphProfile` has been added, enabling the representation of data as a graph with node types, edge types, and foreign key constraints. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to prevent schema truncation in the output context.
