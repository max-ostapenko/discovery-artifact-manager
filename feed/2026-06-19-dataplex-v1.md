# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-19  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with new support for unstructured data profiling and complex graph-based semantic inference.

## Details

The DataScan and DataScanJob resources now support a new `UNSTRUCTURED_DATA_PROFILE` type, accompanied by specific Spec and Result schemas for analyzing non-tabular data. Additionally, a massive set of schemas for Graph Profiling has been introduced, allowing developers to model and retrieve semantic relationships via `GraphProfile`, `NodeType`, and `EdgeType` objects, including support for edge cardinality and foreign key constraints. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to bypass context budget limits when retrieving schema metadata.
