# Dataplex adds unstructured data profiling and graph scans

**Date:** 2026-06-19  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its data scanning capabilities with support for unstructured data profiling and new graph-based semantic inference results.

## Details

The DataScan API now supports a new UNSTRUCTURED_DATA_PROFILE scan type, accompanied by unstructuredDataProfileSpec for configuration and unstructuredDataProfileResult for output. Additionally, a comprehensive set of schemas for graph profiling has been added, including GraphProfile, GraphProfileEdgeType, and GraphProfileNodeType, enabling detailed semantic inference of relationships and entities. Finally, LookupContextRequest now supports an all_schema_fields option to bypass truncation for schema field lists.
