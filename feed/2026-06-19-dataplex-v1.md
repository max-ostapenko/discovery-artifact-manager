# Dataplex adds Unstructured Data Scanning and Graph Profiling

**Date:** 2026-06-19  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex now supports unstructured data profile scans and introduces a comprehensive schema for graph-based semantic inference and relationship mapping.

## Details

The DataScan and DataScanJob resources have been expanded with a new UNSTRUCTURED_DATA_PROFILE type, including specific spec and result schemas (unstructuredDataProfileSpec and unstructuredDataProfileResult). A significant new set of schemas for GraphProfile has been introduced, covering nodeTypes, edgeTypes, and foreignKey constraints to support semantic inference. Additionally, LookupContextRequest now supports an all_schema_fields option, allowing developers to bypass the context_budget truncation for schema metadata.
