# Dataplex adds Unstructured Data Profiling and Graph Metadata

**Date:** 2026-06-18  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its data scanning capabilities with support for unstructured data profiling and a new graph-based semantic inference model.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE type, accompanied by unstructuredDataProfileSpec for configuration and unstructuredDataProfileResult for output. Additionally, a comprehensive set of GraphProfile schemas has been introduced (including nodeTypes, edgeTypes, and foreignKeys), enabling semantic inference scans to map relationships between data entities. Finally, LookupContextRequest adds an all_schema_fields option to bypass truncation in context lookups, ensuring full schema visibility regardless of the context budget.
