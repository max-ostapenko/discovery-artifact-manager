# Dataplex adds Unstructured Data Profiling and Graph Analysis

**Date:** 2026-06-18  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex is expanding its scanning capabilities with a new unstructured data profiling type and a comprehensive graph-based semantic inference model.

## Details

The DataScan API now supports `UNSTRUCTURED_DATA_PROFILE` as a scan type, accompanied by new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. A massive set of schemas for graph profiling has been introduced, including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`, which allow for detailed mapping of relationships, foreign keys, and node types within data. Additionally, the `LookupContextRequest` now includes an `all_schema_fields` option to bypass context budget truncation and return full schema details.
