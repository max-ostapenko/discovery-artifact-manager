# Dataplex adds Unstructured Data Profiling and Graph Schemas

**Date:** 2026-06-18  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new unstructured data profile scan type and introduces a comprehensive schema for graph-based semantic profiling.

## Details

The DataScan and DataScanJob resources now support a new enum value, UNSTRUCTURED_DATA_PROFILE, along with corresponding spec and result fields (unstructuredDataProfileSpec and unstructuredDataProfileResult). Additionally, a large suite of schemas has been added to support GraphProfile, enabling the definition of node types, edge types, foreign key constraints, and extraction hints like cardinality for semantic inference. Finally, LookupContextRequest now includes an all_schema_fields option to force the return of all schema fields regardless of the context budget.
