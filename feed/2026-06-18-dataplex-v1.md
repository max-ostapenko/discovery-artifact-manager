# Dataplex adds Unstructured Data Profiling and Graph Insights

**Date:** 2026-06-18  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex now supports profiling unstructured data and generating graph-based semantic profiles for semantic inference scans.

## Details

This update introduces a major new scan type, UNSTRUCTURED_DATA_PROFILE, available in both DataScan and DataScanJob resources. To support this, a suite of new schemas has been added, including GoogleCloudDataplexV1UnstructuredDataProfileSpec and GoogleCloudDataplexV1GraphProfile, which provides detailed metadata about node types, edge types, and relationship cardinalities (e.g., 1:N) discovered during scans. Additionally, the LookupContextRequest now supports an all_schema_fields option to force the return of all schema fields, bypassing the standard context_budget truncation.
