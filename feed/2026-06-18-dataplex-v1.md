# Dataplex adds Unstructured Data Profiling and Graph Modeling

**Date:** 2026-06-18  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities to include unstructured data and introduces a comprehensive graph profiling schema for semantic inference.

## Details

Dataplex has introduced a major expansion to its DataScan resource. A new scan type, `UNSTRUCTURED_DATA_PROFILE`, allows for profiling non-tabular data sources. Accompanying this is a massive set of new schemas for Graph Profiling (`GraphProfile`, `GraphProfileEdgeType`, `GraphProfileNodeType`), which enables the API to represent complex semantic relationships, including node/edge types, cardinality, and foreign key constraints. Additionally, `LookupContextRequest` now supports an `all_schema_fields` option to ensure full schema retrieval regardless of the context budget.
