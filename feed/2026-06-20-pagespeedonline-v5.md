# PageSpeed Insights adds "Agentic Browsing" category

**Date:** 2026-06-20  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

Google is introducing a new audit category for "agentic browsing" in PageSpeed Insights, likely targeting how AI agents interact with web content.

## Details

The Categories schema has been updated to include a new property, `agentic-browsing`, which maps to a `LighthouseCategoryV5` object. This addition suggests that Lighthouse will soon provide scores and feedback on how well a site supports automated or AI-driven browsing agents, reflecting the rise of LLM-based web navigation.
