# New 'Agentic Browsing' Lighthouse Category Added

**Date:** 2026-06-20  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

PageSpeed Insights now includes a new 'agentic-browsing' category in its Lighthouse reports, likely aimed at evaluating how AI agents interact with web pages.

## Details

The Categories schema has been updated to include a new property: agentic-browsing. This property maps to a LighthouseCategoryV5 object and aggregates audits specifically related to agentic browsing behaviors. This addition suggests Google is beginning to formalize how websites are measured for compatibility with automated AI agents or 'agentic' web crawlers.
