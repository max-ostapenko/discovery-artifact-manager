# New 'Agentic Browsing' category added to PageSpeed reports

**Date:** 2026-06-20  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

PageSpeed Insights has introduced a new 'agentic-browsing' category to its Lighthouse report schema, grouping audits related to automated browsing.

## Details

The API's Categories schema now includes an optional 'agentic-browsing' property of type LighthouseCategoryV5. This addition suggests that Google is beginning to track and report on how well websites accommodate AI agents and automated browsing tools, providing a dedicated set of audits for these use cases.
