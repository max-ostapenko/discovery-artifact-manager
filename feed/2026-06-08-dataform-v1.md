# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-08  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now includes a gitRepositoryLink field, enabling the use of managed machine credentials for Git operations.

## Details

The addition of `gitRepositoryLink` to the `GitRemoteSettings` schema allows developers to point to a specific resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`. This change facilitates integration with Google Cloud's managed connection infrastructure, allowing for more secure, machine-based authentication to external Git repositories without manually handling secrets.
