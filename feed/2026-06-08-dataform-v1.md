# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-08  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports `gitRepositoryLink` in `GitRemoteSettings`, allowing developers to use managed resource names for machine-based Git authentication.

## Details

The `GitRemoteSettings` schema has been updated with a new optional field: `gitRepositoryLink`. This field accepts a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*`. This change facilitates better integration with GCP's managed connection services for handling Git credentials, providing a more secure and structured way to manage machine-to-machine repository access.
