# Iceberg Support for Timestamp Precision Autodetection

**Date:** 2026-05-28  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery has expanded timestamp precision autodetection to Iceberg External Tables and clarified concurrency settings for Python UDFs.

## Details

The `timestampTargetPrecision` field within `ExternalDataConfiguration` and `JobConfigurationLoad` now explicitly includes support for the Iceberg External Table format. This allows for more granular control over how timestamp precisions (e.g., microseconds vs nanoseconds) are handled during autodetection for Iceberg data. Additionally, the description for `containerRequestConcurrency` in `ExternalRuntimeOptions` was updated to specify it applies to Python UDF container instances, providing better context for developers managing remote function performance.
