# New Agentic Browsing Lighthouse Category

**Date:** 2026-06-16  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

PageSpeed Insights has added a new 'agentic-browsing' category to its Lighthouse reports, targeting how AI agents interact with web pages.

## Details

The Categories schema now includes an `agentic-browsing` property, which maps to the `LighthouseCategoryV5` object. This addition allows developers to retrieve scores and audit results specifically focused on 'agentic browsing'—likely measuring how effectively autonomous AI agents or LLM-based browsers can navigate, parse, and interact with a site's content.
