# New Agentic Browsing category added to PageSpeed Insights

**Date:** 2026-06-16  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now includes a new 'agentic-browsing' category for Lighthouse audits, targeting automated and AI-driven browsing scenarios.

## Details

A new field `agentic-browsing` has been added to the `Categories` schema, referencing the `LighthouseCategoryV5` type. This category is designed to house audits related to how web pages perform and behave when accessed by 'agentic' browsers—likely referring to AI agents or automated browsing tools that navigate the web differently than traditional human users.
