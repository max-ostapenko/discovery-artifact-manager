# New Agentic Browsing category added to Lighthouse reports

**Date:** 2026-06-16  
**API:** `pagespeedonline.v5`  
**Impact:** Low  

## Summary

The PageSpeed Insights API now includes a new 'agentic-browsing' category in its Lighthouse results, aggregating audits related to automated agents.

## Details

A new property `agentic-browsing` (of type `LighthouseCategoryV5`) has been added to the `Categories` schema. This addition suggests a shift toward auditing how well web pages accommodate AI agents and automated browsing tools, providing developers with specific metrics on agentic interaction and accessibility.
