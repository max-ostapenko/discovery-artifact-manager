# New Agentic Browsing Category in PageSpeed Insights

**Date:** 2026-06-16  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

PageSpeed Insights adds a new 'agentic-browsing' category to its Lighthouse reports, targeting how AI-driven browsers interact with web pages.

## Details

The API now includes an 'agentic-browsing' property within the Categories schema. This new category maps to the LighthouseCategoryV5 structure and is designed to house audits specifically for agentic browsing scenarios. This indicates a move toward evaluating site performance and accessibility for automated AI agents in addition to human users.
