# Dataplex adds Unstructured Data Scans and Graph Profiling

**Date:** 2026-06-07  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its data scanning capabilities with a new scan type for unstructured data and a comprehensive schema for graph-based semantic inference.

## Details

This update introduces the `UNSTRUCTURED_DATA_PROFILE` scan type to `DataScan` and `DataScanJob` resources, enabling profiling of non-tabular data. Additionally, a large set of schemas for `GraphProfile` has been added, including `GraphProfileNodeType` and `GraphProfileEdgeType`, which allow for detailed mapping of relationships, edge cardinality, and foreign key constraints in data graphs. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to prevent truncation of schema fields in the output context.
