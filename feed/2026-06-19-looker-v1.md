# Looker Core adds Soft Delete recovery and Release Channels

**Date:** 2026-06-19  
**API:** `looker.v1`  
**Impact:** Medium  

## Summary

Looker v1 now supports instance recovery via a new undelete method and provides visibility into soft-deleted instances. Additionally, developers can now configure release channels and accelerated security patches.

## Details

The API introduces a significant lifecycle update with the new instances.undelete method and a showDeleted query parameter for instances.list. The Instance schema has been expanded with softDeleteReason (tracking billing issues, trial expirations, or manual deletions) and suspendedTime. For maintenance control, new fields releaseChannel (RAPID, REGULAR, STABLE) and acceleratedSecurityPatchEnabled have been added to the Instance resource, allowing developers to manage update cadences programmatically.
