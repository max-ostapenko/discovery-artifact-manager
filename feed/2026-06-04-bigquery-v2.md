# Iceberg Support and Python UDF Clarifications

**Date:** 2026-06-04  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery documentation now explicitly includes Iceberg External Tables for timestamp precision autodetection and clarifies Python UDF concurrency settings.

## Details

The timestampTargetPrecision field in both ExternalDataConfiguration and JobConfigurationLoad now lists 'Iceberg External Table' as a supported format for timestamp autodetection. Additionally, the containerRequestConcurrency field in ExternalRuntimeOptions has been clarified to refer specifically to Python UDF container instances rather than generic Cloud Run instances.
