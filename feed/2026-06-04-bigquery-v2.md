# Documentation updates for Iceberg and Python UDFs

**Date:** 2026-06-04  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery updated field descriptions to clarify support for Iceberg External Tables and Python UDF concurrency settings.

## Details

The `timestampTargetPrecision` field in both `ExternalDataConfiguration` and `JobConfigurationLoad` now explicitly mentions support for Iceberg External Tables during autodetection. Additionally, the description for `ExternalRuntimeOptions.containerRequestConcurrency` was updated to specify it applies to Python UDF container instances, providing better clarity for developers tuning remote function performance.
