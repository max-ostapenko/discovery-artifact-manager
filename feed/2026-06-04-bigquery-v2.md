# Expanded Iceberg support and Python UDF clarification

**Date:** 2026-06-04  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery documentation now confirms timestamp precision autodetection support for Iceberg External Tables and clarifies concurrency settings for Python UDFs.

## Details

The timestampTargetPrecision field in both ExternalDataConfiguration and JobConfigurationLoad now explicitly supports Iceberg External Tables alongside CSV, Parquet, and Avro. Additionally, the containerRequestConcurrency field in ExternalRuntimeOptions has been clarified to apply specifically to Python UDF container instances rather than generic Cloud Run instances.
