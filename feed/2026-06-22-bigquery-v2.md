# Clarified transferredBytes for BigQuery Omni

**Date:** 2026-06-22  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

The description for transferredBytes in query statistics has been updated to specifically define its role in BigQuery Omni data movement.

## Details

The transferredBytes field within JobStatistics2 now has a more precise definition. It specifically tracks data movement for BigQuery Omni queries from remote clouds back to Google Cloud (such as query results) over Google-managed connections. The update clarifies that this metric does not include input data read from external data lakes (like S3) when that data stays within the remote cloud.
