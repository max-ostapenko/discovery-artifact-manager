# Clarified transferredBytes definition for BigQuery Omni

**Date:** 2026-06-22  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery has updated the documentation for transferredBytes to clarify that it measures data egress from remote clouds to Google Cloud in Omni queries.

## Details

The update to JobStatistics2.transferredBytes clarifies that this metric tracks data movement over Google-managed connections, such as query results returning to Google Cloud. Crucially, it notes that input data read from external data lakes (like S3) is not included in this count as that data stays within the remote cloud. This provides better visibility into what exactly is being measured during cross-cloud analytics.
