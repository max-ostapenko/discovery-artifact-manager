# Dataplex Data Domains add IAM policy member metadata

**Date:** 2026-07-02  
**API:** `dataplex.v1`  
**Impact:** Low  

## Summary

The recently introduced DataDomain resource now includes output-only IAM policy member strings, supporting both name-based and UID-based identity principals.

## Details

The GoogleCloudDataplexV1DataDomain schema now includes a policyMember field. This field maps to a new GoogleIamV1ResourcePolicyMember object containing iamPolicyNamePrincipal (AIP-122 compliant name-based principal) and iamPolicyUidPrincipal (AIP-148 compliant system-assigned unique identifier). These fields allow developers to programmatically identify the specific IAM identities associated with a Data Domain, facilitating more robust security auditing and access management.
