# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-21  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform now supports referencing managed GitRepositoryLink resources for machine-based authentication to remote repositories.

## Details

The GitRemoteSettings schema has been updated to include a new optional field, gitRepositoryLink. This field allows developers to specify a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*, enabling the use of machine credentials via Google Cloud's managed connection infrastructure rather than relying solely on manual secret management.
