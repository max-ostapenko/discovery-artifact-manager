# Dataform adds support for managed Git repository links

**Date:** 2026-06-21  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform now supports the `gitRepositoryLink` field in Git settings, enabling the use of managed machine credentials for repository connections.

## Details

The `GitRemoteSettings` schema has been updated to include an optional `gitRepositoryLink` string. This field allows developers to reference a managed connection resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) instead of manually managing secrets or SSH keys within Dataform. This aligns Dataform with GCP's broader move toward centralized connection management.
