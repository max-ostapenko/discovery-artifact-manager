# Dataform adds managed Git repository links

**Date:** 2026-06-21  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's Git settings now support gitRepositoryLink, enabling the use of managed machine credentials via GCP connection resources.

## Details

A new optional field gitRepositoryLink has been added to the GitRemoteSettings schema. This field allows developers to point to a specific Git connection resource (format: projects/*/locations/*/connections/*/gitRepositoryLinks/*) for authentication. This update facilitates a more secure and standardized way to handle Git authentication by leveraging GCP's managed connection infrastructure instead of manual secret management.
