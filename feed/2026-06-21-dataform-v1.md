# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-21  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform's GitRemoteSettings now supports a gitRepositoryLink field, enabling more secure, managed Git connections.

## Details

The GitRemoteSettings schema has been updated to include a new optional string field: gitRepositoryLink. This field allows developers to specify a resource name (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*) to handle machine credentials. This likely integrates with GCP's Developer Connect service to streamline how Dataform authenticates with remote repositories without manual secret handling.
