# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-21  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's Git settings now support gitRepositoryLink, allowing developers to link to specific repository resources for machine-based authentication.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field accepts a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*, enabling more robust integration with Google Cloud's repository connection framework for automated workflows and machine credentials.
