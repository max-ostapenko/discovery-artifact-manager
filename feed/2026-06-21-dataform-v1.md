# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-21  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now supports a gitRepositoryLink field for centralized machine credential management.

## Details

The GitRemoteSettings schema has been updated to include a new optional string field: gitRepositoryLink. This field accepts a resource name following the pattern projects/*/locations/*/connections/*/gitRepositoryLinks/*. This addition facilitates the use of managed Git connections (likely via Developer Connect) for machine-based authentication, moving away from manual secret handling for repository access.
