# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-21  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking Git repository resources for machine credentials, simplifying authentication for automated workflows.

## Details

A new optional field `gitRepositoryLink` has been added to the `GitRemoteSettings` schema. This allows developers to point to a managed Git repository resource (format: `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) instead of manually managing secrets for machine-based Git operations. This likely integrates with Google Cloud's Developer Connect for more secure and centralized credential management.
