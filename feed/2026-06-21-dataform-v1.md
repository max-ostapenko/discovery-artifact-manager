# Dataform adds support for GitRepositoryLink credentials

**Date:** 2026-06-21  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform now supports linking to managed GitRepositoryLink resources for machine credentials, streamlining Git authentication.

## Details

The GitRemoteSettings resource now includes an optional gitRepositoryLink field. This allows users to reference a managed Git connection (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*) instead of relying solely on manual credential configurations. This change facilitates better integration with Google Cloud's native Git connection management services for machine-to-machine authentication.
