# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-21  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now supports a gitRepositoryLink field for managing machine credentials via resource names.

## Details

A new optional field `gitRepositoryLink` has been added to the `GitRemoteSettings` schema. This allows developers to specify a resource name in the format `projects/*/locations/*/connections/*/gitRepositoryLinks/*` to handle machine credentials, likely integrating with Google Cloud's Developer Connect or similar centralized connection services.
