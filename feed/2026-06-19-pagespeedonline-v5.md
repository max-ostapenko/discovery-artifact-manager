# PageSpeed Insights adds Agentic Browsing category

**Date:** 2026-06-19  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

A new 'agentic-browsing' category has been added to the PageSpeed Insights (Lighthouse) report schema, signaling new audits for AI-driven browsing.

## Details

The Categories schema now includes an 'agentic-browsing' property of type LighthouseCategoryV5. This addition allows the API to return a specific score and set of audit results focused on agentic browsing, likely reflecting a shift toward optimizing web content for non-human visitors and AI agents.
