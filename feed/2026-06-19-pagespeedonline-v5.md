# PageSpeed Insights adds 'Agentic Browsing' category

**Date:** 2026-06-19  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The API now returns a new 'agentic-browsing' category in Lighthouse reports, signaling a shift toward auditing how AI agents interact with web content.

## Details

The Categories schema has been updated to include a new property, agentic-browsing, which maps to a LighthouseCategoryV5 object. This new category is designed to house audits specifically related to agentic browsing behavior, allowing developers to measure and optimize how automated or AI-driven agents experience their web pages alongside traditional metrics like performance and accessibility.
