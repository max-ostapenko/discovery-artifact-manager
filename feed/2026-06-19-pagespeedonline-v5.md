# New Agentic Browsing Category in PageSpeed Insights

**Date:** 2026-06-19  
**API:** `pagespeedonline.v5`  
**Impact:** Medium  

## Summary

The PageSpeed Insights API now includes a new 'agentic-browsing' category in Lighthouse reports, focusing on audits related to automated or AI-driven browsing.

## Details

A new field 'agentic-browsing' has been added to the 'Categories' schema, referencing 'LighthouseCategoryV5'. This addition suggests that Google is beginning to measure how well websites perform or behave when accessed by 'agentic' browsers—likely AI agents or advanced automated crawlers. Developers can expect to see new audit data within this category in API responses, providing insights into how machine-driven users experience their pages.
