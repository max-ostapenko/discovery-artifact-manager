# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-12  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform now supports linking GitRepositoryLink resources for machine-based authentication in Git remote settings.

## Details

The `GitRemoteSettings` schema has been expanded with a new optional field: `gitRepositoryLink`. This field allows developers to specify a resource name (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) to handle machine credentials. This provides a more structured way to manage Git authentication compared to traditional token-based setups.
