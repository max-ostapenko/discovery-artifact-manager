# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-12  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform now supports linking to Git repositories via the GitRepositoryLink resource, enabling more streamlined machine credential management.

## Details

The GitRemoteSettings schema has been updated with a new optional field: gitRepositoryLink. This field allows developers to specify a resource name in the format projects/*/locations/*/connections/*/gitRepositoryLinks/*, facilitating the use of managed connections for Git authentication instead of manual credential handling.
