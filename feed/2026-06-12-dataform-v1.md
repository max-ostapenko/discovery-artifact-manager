# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-12  
**API:** `dataform.v1`  
**Impact:** Low  

## Summary

Dataform's GitRemoteSettings now supports a gitRepositoryLink field, enabling more streamlined machine credential management via GCP connection resources.

## Details

The GitRemoteSettings schema now includes an optional `gitRepositoryLink` field. This allows developers to point to a specific GitRepositoryLink resource (formatted as `projects/*/locations/*/connections/*/gitRepositoryLinks/*`) to handle machine-based authentication. This is a cleaner alternative to manual credential handling for automated Dataform workflows and likely integrates with the broader GCP Developer Connect ecosystem.
