# Dataform adds GitRepositoryLink for machine credentials

**Date:** 2026-06-12  
**API:** `dataform.v1`  
**Impact:** Medium  

## Summary

Dataform's GitRemoteSettings now supports gitRepositoryLink, enabling more streamlined machine-based authentication via centralized repository links.

## Details

The GitRemoteSettings schema has been updated to include a new optional field: gitRepositoryLink. This field allows developers to specify a resource name (formatted as projects/*/locations/*/connections/*/gitRepositoryLinks/*) to be used for machine credentials, likely simplifying how Dataform interacts with remote Git providers in automated environments.
