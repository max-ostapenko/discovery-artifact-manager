# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-16  
**API:** `dataplex.v1`  
**Impact:** High  

## Summary

Dataplex expands its data scanning capabilities with support for unstructured data profiling and semantic graph inference.

## Details

The DataScan and DataScanJob resources now support a new scan type: UNSTRUCTURED_DATA_PROFILE, accompanied by specific Spec and Result fields for managing these scans. Additionally, a massive set of schemas for Graph Profiling has been introduced (including GraphProfile, GraphProfileEdgeType, and GraphProfileNodeType), allowing developers to model and extract semantic relationships, foreign keys, and cardinality from their data. Finally, the LookupContextRequest now supports an all_schema_fields option to bypass character budget truncation for schema fields.
