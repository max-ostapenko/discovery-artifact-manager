# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-16  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex v1 introduces support for unstructured data profile scans and a comprehensive graph profiling schema for semantic inference.

## Details

This update introduces `UNSTRUCTURED_DATA_PROFILE` as a new scan type within `DataScan` and `DataScanJob` resources, supported by the new `UnstructuredDataProfileSpec` and `UnstructuredDataProfileResult` schemas. Additionally, a significant set of schemas has been added to support graph-based semantic inference, including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`, which allow for mapping relationships and extraction hints in data graphs. Finally, `LookupContextRequest` now supports an `all_schema_fields` option to bypass character budget limits when retrieving schema metadata.
