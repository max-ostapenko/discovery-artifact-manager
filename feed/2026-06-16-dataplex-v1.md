# Dataplex adds Unstructured Data Scans and Graph Profiling

**Date:** 2026-06-16  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its profiling capabilities with a new scan type for unstructured data and a comprehensive graph-based semantic inference model.

## Details

The DataScan and DataScanJob resources now support a new enum value, UNSTRUCTURED_DATA_PROFILE, accompanied by specific Spec and Result schemas for profiling non-tabular data. Additionally, a significant new GraphProfile subsystem has been introduced, including schemas for NodeTypes, EdgeTypes, and ForeignKeys, which allows Dataplex to model and infer semantic relationships across data assets. Finally, the LookupContextRequest now includes an all_schema_fields option to prevent the truncation of schema fields in the returned context.
