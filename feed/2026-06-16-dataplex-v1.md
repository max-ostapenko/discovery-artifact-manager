# Dataplex adds Unstructured Data Profiling and Graph Metadata

**Date:** 2026-06-16  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex is expanding its scanning capabilities with support for unstructured data profiling and a new graph-based semantic inference system.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE type, accompanied by specific Spec and Result schemas for handling non-tabular data. Additionally, a massive set of schemas for GraphProfile has been introduced, allowing developers to work with inferred graph metadata including nodeTypes, edgeTypes, and relationship cardinalities. Finally, LookupContextRequest now supports an all_schema_fields flag to bypass context budget truncation for schema fields.
