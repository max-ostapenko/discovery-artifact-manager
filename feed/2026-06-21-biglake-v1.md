# BigLake API Rebrands to Lakehouse API

**Date:** 2026-06-21  
**API:** `biglake.v1`  
**Impact:** Low  

## Summary

The BigLake API has been rebranded to the Lakehouse API, signaling a shift toward a multi-engine, cross-cloud data lakehouse strategy.

## Details

The API description has been updated to reflect its new identity as the Lakehouse API. It now emphasizes support for multiple engines including Apache Spark, Flink, and Trino, alongside BigQuery. The update also highlights 'Lakehouse runtime catalog federation,' which enables discovery and analysis of data across Snowflake, Databricks, and AWS Glue. Despite the significant shift in product positioning, there are no changes to the actual API methods or schema in this update.
