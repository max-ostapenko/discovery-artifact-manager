# Dataplex adds Unstructured Data Profiling and Graph Inference

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new scan type for unstructured data and a comprehensive GraphProfile schema for semantic relationship mapping.

## Details

The DataScan and DataScanJob resources now support a new `UNSTRUCTURED_DATA_PROFILE` type, accompanied by `unstructuredDataProfileSpec` for configuration and `unstructuredDataProfileResult` for output. A significant set of new schemas has been introduced to support graph-based semantic inference, including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`, allowing developers to model relationships, cardinalities, and foreign keys within their data fabric. Additionally, `LookupContextRequest` now includes an `all_schema_fields` option to bypass truncation and return all schema fields in the context.
