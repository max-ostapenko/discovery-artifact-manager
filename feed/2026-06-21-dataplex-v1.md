# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with support for unstructured data profiling and a new graph-based profiling schema for semantic inference.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE type, accompanied by specific Spec and Result fields for managing scans on non-tabular data. Additionally, a comprehensive set of GraphProfile schemas has been introduced (including EdgeType and NodeType), enabling semantic inference scans that map relationships and entities within data lakes. Finally, LookupContextRequest now supports an all_schema_fields boolean to prevent truncation of schema metadata in the response.
