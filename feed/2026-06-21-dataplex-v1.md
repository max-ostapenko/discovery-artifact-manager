# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** High  

## Summary

Dataplex expands its scanning capabilities with a new scan type for unstructured data and a comprehensive suite of Graph Profiling tools for semantic inference.

## Details

The DataScan resource now supports a new enum value, UNSTRUCTURED_DATA_PROFILE, enabling profiling of non-tabular data sources. Accompanying this is a significant addition of graph-related schemas, including GoogleCloudDataplexV1GraphProfile, GraphProfileEdgeType, and GraphProfileNodeType, which facilitate semantic inference by mapping relationships, edge cardinality, and foreign key constraints. Additionally, LookupContextRequest now includes an all_schema_fields option to bypass character budget truncation when retrieving schema metadata.
