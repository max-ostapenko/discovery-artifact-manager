# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with support for unstructured data profiling and a new graph-based semantic inference model.

## Details

The DataScan and DataScanJob resources now support a new scan type: UNSTRUCTURED_DATA_PROFILE. This includes the addition of unstructuredDataProfileSpec for configuration and unstructuredDataProfileResult for output. Furthermore, a large set of schemas for GraphProfile has been introduced (including EdgeTypes and NodeTypes), enabling semantic inference and relationship mapping within data scans. Finally, the LookupContextRequest now supports an all_schema_fields option to bypass character budget truncation for schema fields.
