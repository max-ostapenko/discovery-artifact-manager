# Dataplex adds Unstructured Data Profiling and Graph Profiles

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new scan type for unstructured data and introduces a comprehensive GraphProfile schema for semantic inference.

## Details

The DataScan and DataScanJob resources now support a new `UNSTRUCTURED_DATA_PROFILE` type, accompanied by `unstructuredDataProfileSpec` for configuration and `unstructuredDataProfileResult` for output. A significant new schema set for `GraphProfile` has been introduced, allowing developers to model complex data relationships including `nodeTypes`, `edgeTypes`, and `foreignKeys`. Additionally, `LookupContextRequest` now includes an `all_schema_fields` option to force the return of all schema fields, bypassing the standard `context_budget` truncation.
