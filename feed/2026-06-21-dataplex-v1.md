# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex DataScans now support a new UNSTRUCTURED_DATA_PROFILE type and a comprehensive GraphProfile schema for semantic inference.

## Details

Developers can now configure DataScan jobs with the new UNSTRUCTURED_DATA_PROFILE enum, supported by the addition of unstructuredDataProfileSpec and unstructuredDataProfileResult schemas. Furthermore, a significant set of GraphProfile schemas has been introduced (including nodeTypes, edgeTypes, and foreignKeys), enabling the API to represent complex semantic relationships and graph structures inferred from data. Additionally, the LookupContextRequest now supports an all_schema_fields option to force the return of all schema fields regardless of the defined context budget.
