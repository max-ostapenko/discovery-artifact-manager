# Dataplex adds Unstructured Data Profiling and Graph Analysis

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** High  

## Summary

Dataplex expands its scanning capabilities with support for unstructured data profiling and a new graph-based semantic inference engine.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE type, accompanied by unstructuredDataProfileSpec for configuration and unstructuredDataProfileResult for output. Additionally, a comprehensive set of Graph Profile schemas has been introduced (including GraphProfile, GraphProfileNodeType, and GraphProfileEdgeType), enabling detailed semantic inference of data relationships and connectivity topology. Finally, LookupContextRequest now supports an all_schema_fields option to bypass context budget truncation for schema fields, ensuring full schema visibility when needed.
