# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex introduces a new scan type for unstructured data and a comprehensive graph profiling system for semantic inference.

## Details

Dataplex is expanding its DataScan capabilities with the addition of the `UNSTRUCTURED_DATA_PROFILE` scan type, supported by new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. Furthermore, a massive new schema set for `GraphProfile` has been introduced, allowing developers to model and retrieve semantic relationships via `nodeTypes` and `edgeTypes`, including foreign key constraints and cardinality hints. Additionally, `LookupContextRequest` now supports an `all_schema_fields` option to bypass truncation when retrieving metadata.
