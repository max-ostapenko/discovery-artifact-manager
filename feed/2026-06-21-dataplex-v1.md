# Dataplex adds Unstructured Data and Graph Profiling

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex DataScans now support unstructured data profiling and a new Graph Profile suite for semantic inference of data relationships.

## Details

This update introduces the `UNSTRUCTURED_DATA_PROFILE` scan type to `DataScan` and `DataScanJob` resources, accompanied by new `unstructuredDataProfileSpec` and `unstructuredDataProfileResult` fields. Furthermore, a massive set of schemas for Graph Profiling has been added (including `GraphProfile`, `GraphProfileEdgeType`, and `GraphProfileNodeType`), enabling developers to model and extract semantic relationships, node types, and edge cardinalities from their data. Additionally, `LookupContextRequest` now supports an `all_schema_fields` option to ensure all fields are returned regardless of the character budget.
