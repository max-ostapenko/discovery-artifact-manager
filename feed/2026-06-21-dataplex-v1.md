# Dataplex adds unstructured data profiling and graph profiles

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with support for unstructured data profiling and a new graph profiling schema for semantic inference.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE type, accompanied by unstructuredDataProfileSpec for configuration and unstructuredDataProfileResult for output. Additionally, a comprehensive set of schemas for GraphProfile has been introduced—including GraphProfileEdgeType and GraphProfileNodeType—enabling detailed semantic inference of graph-based data structures. Finally, LookupContextRequest now supports an all_schema_fields option to bypass truncation when retrieving schema metadata, ensuring full visibility into data structures regardless of the context budget.
