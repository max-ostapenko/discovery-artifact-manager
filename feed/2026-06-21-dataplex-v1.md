# Dataplex adds Unstructured Data Profiling and Graph Metadata

**Date:** 2026-06-21  
**API:** `dataplex.v1`  
**Impact:** Medium  

## Summary

Dataplex expands its scanning capabilities with a new unstructured data profile type and a comprehensive schema set for graph-based semantic inference.

## Details

The DataScan and DataScanJob resources now support a new UNSTRUCTURED_DATA_PROFILE scan type, accompanied by specific Spec and Result objects for profiling non-tabular data. Additionally, a large suite of GraphProfile schemas has been introduced (including NodeTypes, EdgeTypes, and ForeignKeys), enabling Dataplex to model complex semantic relationships and cardinalities (e.g., 1:N) across data assets. Finally, LookupContextRequest now supports an all_schema_fields option to bypass truncation and return full schema metadata.
