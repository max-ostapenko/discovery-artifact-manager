# Expanded Iceberg support and clarified UDF concurrency

**Date:** 2026-06-01  
**API:** `bigquery.v2`  
**Impact:** Low  

## Summary

BigQuery updated documentation to reflect support for Iceberg External Tables in timestamp precision settings and clarified Python UDF container concurrency.

## Details

The `timestampTargetPrecision` field in both `ExternalDataConfiguration` and `JobConfigurationLoad` now explicitly supports the Iceberg External Table format for autodetection mode. Additionally, the `containerRequestConcurrency` field in `ExternalRuntimeOptions` has been clarified to refer specifically to Python UDF container instances rather than generic Cloud Run instances, providing better context for developers managing remote function performance.
